# L'accord du participe passé

## 1. La question, en une phrase

> Le participe passé s'accorde — ou non — selon **l'auxiliaire** et selon la **place du complément d'objet direct (COD)**. Avec ***être***, il s'accorde toujours avec le **sujet**. Avec ***avoir***, il ne s'accorde **qu'avec un COD placé avant le verbe** (COD *antéposé*) ; sinon, il reste invariable. Les verbes **pronominaux** suivent une troisième logique, dérivée des deux premières.

C'est le piège le plus rentable du B1 : à l'écrit, une erreur d'accord se voit immédiatement, et le TCF la teste très souvent (relatives en *que*, pronoms COD antéposés, pronominaux). La bonne nouvelle : trois règles bien posées couvrent la quasi-totalité des cas.

## 2. Les trois cas de base

### 2.1 Auxiliaire *être* → accord avec le SUJET

Le participe se comporte comme un adjectif attribut : il prend le genre et le nombre du **sujet**. Cela vaut pour les verbes de mouvement conjugués avec *être* et pour tous les passifs.

| Sujet | Exemple | Terminaison |
|---|---|---|
| masc. sing. | *Il est **parti**.* | — |
| fém. sing. | *Elle est **partie**.* | *-e* |
| masc. plur. | *Ils sont **arrivés**.* | *-s* |
| fém. plur. | *Elles sont **arrivées**.* | *-es* |
| passif (fém. plur.) | *Les lois ont été **votées**.* | *-es* |

> *Mes sœurs sont **nées** à Lyon.* — sujet fém. plur. → *-es*.
> *La réforme a été **adoptée** hier.* — passif, sujet fém. sing. → *-e*.

<!-- domain: politics -->

### 2.2 Auxiliaire *avoir* → accord avec le COD ANTÉPOSÉ seulement

Avec *avoir*, le participe reste **invariable par défaut**. Il s'accorde **uniquement** si un COD est placé **avant** le verbe. Ce COD antéposé est le plus souvent :
- le pronom relatif ***que*** ;
- un pronom personnel COD (*le, la, les, me, te, nous, vous*) ;
- un mot interrogatif ou exclamatif (*quelle erreur…*, *combien de lettres…*).

| Situation | Exemple | Accord ? |
|---|---|---|
| COD **après** le verbe | *J'ai **acheté** des fleurs.* | non |
| pas de COD | *Elle a **mangé**.* | non |
| COD = *que* antéposé | *Les fleurs que j'ai **achetées***. | oui (fém. plur.) |
| COD = pronom antéposé | *Je les ai **vues**.* | oui (= *les*, fém. plur.) |
| COD interrogatif/exclamatif antéposé | *Quelle erreur a-t-il **commise** !* | oui (fém. sing.) |
| complément = **COI** | *Je leur ai **parlé**.* | non |
| complément repris par ***en*** | *Des pommes, j'**en** ai **mangé**.* | non |

> **Test décisif** : posez la question « **qui ? / quoi ?** » après le verbe. Si la réponse est **déjà placée avant** le participe, on accorde. Sinon, on n'accorde pas.

> ⚠ **Pas d'accord avec un COI** ni avec ***en***. *Je **leur** ai parlé* (*leur* = *à eux*, COI → invariable) ; *Des gâteaux, j'**en** ai mangé* (*en* reprend un partitif, jamais traité comme COD antéposé). Détail relié dans `gram-b1-07`.

### 2.3 Verbes pronominaux → deux sous-règles

Les pronominaux se conjuguent avec *être*, mais leur accord obéit en réalité à la **logique du COD** (comme avec *avoir*), parce que le pronom réfléchi peut être COD **ou** COI.

| Type | Test | Accord ? | Exemple |
|---|---|---|---|
| essentiellement pronominal | n'existe qu'à la forme pronominale | avec le **sujet** | *Elle s'est **souvenue**.* |
| accidentellement pronominal, *se* = COD | « elle a lavé **qui ?** → se » | avec le **sujet** | *Elle s'est **lavée**.* |
| accidentellement pronominal, *se* = COI | un autre COD suit | **invariable** | *Elle s'est **lavé** les mains.* |
| réciproque / *se* = COI | *parler/téléphoner/écrire **à*** qqn | **invariable** | *Ils se sont **parlé**.* |

> **Essentiellement pronominaux** (pas de forme sans *se*) : *s'enfuir, se souvenir, s'évanouir, s'absenter, se méfier…* → accord avec le sujet.
> *Elle s'est **enfuie**.* / *Ils se sont **méfiés**.*

## 3. Contrastes à mémoriser

### 3.1 COD antéposé vs COD postposé (auxiliaire *avoir*)

> *J'ai **écrit** trois lettres.* — COD *trois lettres* **après** le verbe → invariable.
> *Les trois lettres que j'ai **écrites**.* — COD *que* (= *les lettres*) **avant** le verbe → *-es*.

C'est l'opposition centrale de toute la leçon. La même phrase change d'orthographe selon la **place** du COD, pas selon son sens.

<!-- domain: work -->

### 3.2 COD antéposé (accord) vs COI (pas d'accord)

> *Mes collègues ? Je les ai **prévenus** ce matin.* — *les* = COD antéposé (*prévenir qqn*) → accord.
> *Mes collègues ? Je leur ai **parlé** ce matin.* — *leur* = COI (*parler à qqn*) → invariable.

**Test** : si le verbe se construit *verbe + à qqn*, le pronom est COI et il n'y a **pas** d'accord. (Voir les verbes-pièges de `gram-b1-07` : *téléphoner, répondre, parler, plaire, obéir…*)

### 3.3 COD antéposé (accord) vs reprise par *en* (pas d'accord)

> *Ces articles, je les ai **lus**.* — *les* = COD antéposé → accord.
> *Des articles, j'**en** ai **lu** plusieurs.* — *en* reprend un quantifié indéfini → invariable.

> ⚠ ***En*** ne déclenche **jamais** l'accord, même quand il renvoie à un féminin pluriel : *Des pommes, j'en ai **mangé** trois.* (et non \*« mangées »).
> <!-- AUDIT: cas standard (Riegel ; Grevisse). Quelques grammairiens tolèrent l'accord quand "en" est précisé par un adverbe de quantité ; la norme scolaire et le TCF retiennent l'invariabilité — règle à confirmer pour les cas marginaux. -->

### 3.4 Pronominal *se* = COD (accord) vs *se* = COI + COD postposé (pas d'accord)

> *Elle s'est **lavée**.* — *se* = COD (elle a lavé **elle-même**) → accord avec le sujet.
> *Elle s'est **lavé** les mains.* — *se* = COI (à elle-même), le COD *les mains* est **postposé** → invariable.
> *Les mains qu'elle s'est **lavées**.* — ici *que* (= *les mains*) est COD **antéposé** → accord !

C'est le contraste le plus subtil du chapitre : un COD **postposé** (*les mains*) bloque l'accord, mais le même COD **antéposé** (*que*) le rétablit.

### 3.5 Réciproque *se* = COI (pas d'accord) vs *se* = COD (accord)

> *Ils se sont **parlé**.* / *Elles se sont **téléphoné**.* / *Ils se sont **écrit**.* — *parler/téléphoner/écrire **à** qqn* → *se* = COI → invariable.
> *Ils se sont **rencontrés**.* / *Elles se sont **vues**.* — *rencontrer/voir **qqn*** (construction directe) → *se* = COD → accord.

**Test** : remplacez par un verbe à construction directe. *Ils se sont battus* (*battre qqn*, COD → accord) vs *Ils se sont nui* (*nuire à qqn*, COI → invariable).

### 3.6 Participe + infinitif, et les cas *fait* / *laissé*

C'est la zone fine du B1 ; gardons l'essentiel.

> *Je les ai **vus** partir.* — *les* est COD de *voir* **et** sujet de *partir* (ce sont eux qui partent) → accord.
> *Les airs que j'ai **entendu** chanter.* — *que* (= *les airs*) est l'objet de *chanter*, **pas** le sujet (les airs ne chantent pas) → invariable.

**Règle** : devant un infinitif, le participe (avec *avoir*) s'accorde si le COD antéposé est le **sujet** de l'action exprimée par l'infinitif ; sinon, il reste invariable.

> ⚠ ***fait* + infinitif est TOUJOURS invariable** : *Les robes qu'elle a **fait** faire.* / *Elle s'est **fait** couper les cheveux.* (*fait* ne s'accorde jamais devant un infinitif.)
> <!-- AUDIT: invariabilité de "fait + infinitif" — règle stable (Riegel ; Grevisse ; rectifications de 1990). -->

> ⚠ ***laissé* + infinitif** : la règle moderne (rectifications de 1990, reprise par l'OQLF) recommande de **laisser *laissé* invariable** devant un infinitif, comme *fait* : *Elle s'est **laissé** convaincre.* / *Les enfants qu'elle a **laissé** jouer.*
> <!-- AUDIT: "laissé + infinitif" invariable selon les rectifications de 1990 et la BDL/OQLF ; l'accord traditionnel (laissé variable quand l'antécédent est sujet de l'infinitif) reste toléré. Présentation simplifiée pour le B1 — à confirmer en revue native. -->

## 4. Faux amis et confusions à éviter

### 4.1 *être* « de mouvement » ≠ *avoir*

Certains verbes hésitent entre les deux auxiliaires selon le sens, ce qui change l'accord :

> *Elle est **montée** au troisième étage.* — *monter* intransitif → *être* → accord avec le sujet.
> *Elle a **monté** les valises.* — *monter qqch* (transitif) → *avoir* → COD *les valises* postposé → invariable.
> *Les valises qu'elle a **montées**.* — même verbe, COD *que* antéposé → accord.

Même logique pour *descendre, sortir, rentrer, retourner, passer*.

### 4.2 *-é*, *-er* ou *-ez* ? (le piège phonétique)

À l'oral, *-é*, *-er* et *-ez* se prononcent pareil. Test : remplacez par un verbe du 3ᵉ groupe (*prendre / pris*).

> *J'ai **mangé**.* → *J'ai **pris*** → participe passé, donc *-é* (*mangé*).
> *Je vais **manger**.* → *Je vais **prendre*** → infinitif, donc *-er* (*manger*).

### 4.3 Accord du participe ≠ accord de l'adjectif

> *La porte est **ouverte**.* — *ouverte* peut être participe (passif : *on a ouvert la porte*) **ou** adjectif (état). Dans les deux cas, accord avec *la porte*.
> *La porte, je l'ai **ouverte**.* — ici clairement participe + COD antéposé → accord avec *l'* (= *la porte*).

Le résultat est le même, mais la justification diffère : ne pas mélanger « accord avec le sujet » (être/adjectif) et « accord avec le COD antéposé » (avoir).

### 7.2 MCQ

- Q1 → **(c) prises** — COD *que* (= *les photos*, fém. plur.) antéposé, aux. *avoir* → accord.
- Q2 → **(a) lavé** — *se* = COI, COD *les cheveux* postposé → invariable.
- Q3 → **(c) les / classés** — *les* = COD antéposé (*classer qqch*) → accord (*classés*, masc. plur.) ; *leur* serait un COI, ici incorrect.

### 7.3 Transformation / accord

1. *Les chercheuses sont **revenues** de la conférence…* — aux. *être* → accord avec le sujet (fém. plur.).
2. *La proposition que la directrice a **faite**…* — COD *que* (= *la proposition*, fém. sing.) antéposé → accord.
3. *Elle s'est **préparé** un café, puis elle s'est **assise** près de la fenêtre.* — *préparé* : *se* = COI, COD *un café* postposé → invariable ; *assise* : *s'asseoir*, *se* = COD → accord avec le sujet.
4. *Des solutions, nous en avons **envisagé** plusieurs, mais celles que nous avons **retenues** sont prudentes.* — *envisagé* : reprise par *en* → invariable ; *retenues* : COD *que* (= *celles*, fém. plur.) antéposé → accord.

## 8. Carte de synthèse

**L'accord du participe passé — l'essentiel**

| Auxiliaire / type | On accorde avec… | Condition |
|---|---|---|
| ***être*** (mouvement, passif) | le **sujet** | toujours |
| ***avoir*** | le **COD** | seulement s'il est **antéposé** |
| pronominal, *se* = **COD** | le **sujet** | *Elle s'est **lavée*** |
| pronominal, *se* = **COI** | — | invariable (*Elle s'est **lavé** les mains*) |
| essentiellement pronominal | le **sujet** | *Elle s'est **souvenue*** |

**Avec *avoir* — le réflexe en deux temps**
1. Y a-t-il un COD ? (« qui ? / quoi ? » après le verbe)
2. Est-il **placé avant** le verbe ? → oui : accord ; non / COI / *en* : invariable.

**Toujours invariables**
- complément **COI** (*je leur ai parlé*) ;
- reprise par ***en*** (*j'en ai mangé*) ;
- ***fait* + infinitif** (*les robes qu'elle a fait faire*) ;
- ***laissé* + infinitif** (rectifications 1990 : *elle s'est laissé convaincre*).

**Test phonétique *-é / -er*** : remplacer par *pris* (participe) ou *prendre* (infinitif).
