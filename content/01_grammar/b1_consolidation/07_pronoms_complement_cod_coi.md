# Pronoms personnels compléments (COD / COI) et leur ordre

## 1. La question, en une phrase

> Un pronom complément **remplace un groupe nominal déjà connu** pour éviter de le répéter. Le choix de la forme dépend de la construction du verbe : complément **direct** (*verbe + qqch / qqn*, sans préposition) → COD ; complément **indirect introduit par *à*** (*verbe + à qqn*) → COI. La place et l'ordre de ces pronoms obéissent à des règles fixes.

C'est un palier de fluidité du B1 : savoir dire *je la connais* / *je lui parle* sans hésiter, et enchaîner *je le lui ai donné*. Au TCF, ces pronoms sont omniprésents, à l'oral comme à l'écrit.

> Les pronoms ***en*** et ***y*** (qui remplacent *de + qqch* et *à + qqch / un lieu*) font l'objet de l'unité `gram-b1-08` ; on ne les détaille pas ici, mais ils apparaissent dans la grille d'ordre du §2.

## 2. Les formes

### 2.1 COD vs COI : les deux séries

| Personne | COD (*verbe + qqch/qqn*) | COI (*verbe + à qqn*) |
|---|---|---|
| 1ʳᵉ sing. | **me** (m') | **me** (m') |
| 2ᵉ sing. | **te** (t') | **te** (t') |
| 3ᵉ sing. | **le** (m.) / **la** (f.) (l') | **lui** (m. et f.) |
| 1ʳᵉ plur. | **nous** | **nous** |
| 2ᵉ plur. | **vous** | **vous** |
| 3ᵉ plur. | **les** | **leur** |

> Les formes ne diffèrent **qu'à la 3ᵉ personne** : *le / la / les* (COD) face à *lui / leur* (COI). Aux autres personnes, *me, te, nous, vous* servent pour les deux fonctions. Toute la difficulté du B1 se concentre donc sur la 3ᵉ personne.

> ⚠ *lui* est **invariable en genre** : *Je parle à Marc → je **lui** parle* ; *Je parle à Marie → je **lui** parle*. Il n'y a pas de forme féminine \*« leur elle ».

### 2.2 La grille d'ordre des doubles pronoms

Quand deux pronoms compléments précèdent le verbe, ils suivent toujours cet ordre figé :

| 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|
| me | le | lui | y | en |
| te | la | leur |  |  |
| se | les |  |  |  |
| nous |  |  |  |  |
| vous |  |  |  |  |

> Lecture de gauche à droite : *me / te / se / nous / vous* **avant** *le / la / les* **avant** *lui / leur* **avant** *y* **avant** *en*.
> *Il **me le** donne.* (colonne 1 + colonne 2)
> *Je **le lui** explique.* (colonne 2 + colonne 3)
> *Il **m'en** parle.* (colonne 1 + colonne 5)

> ⚠ Conséquence pratique : on dit *je **le lui** donne* (COD avant COI à la 3ᵉ personne), mais *il **me le** donne* (COI avant COD aux 1ʳᵉ/2ᵉ personnes). L'ordre n'est donc **pas** « toujours COI avant COD » ; il suit la grille.

## 3. Les six contrastes à mémoriser

### 3.1 Construction directe (COD) vs construction indirecte (COI)

> *J'aide **mon collègue**.* → *Je **l'**aide.* — *aider qqn* = COD.
> *Je téléphone **à mon collègue**.* → *Je **lui** téléphone.* — *téléphoner à qqn* = COI.

**Test décisif** : on remplace le complément par « **quelque chose / quelqu'un** » (sans préposition) → COD ; par « **à quelqu'un** » → COI. La préposition *à* est le signal du COI.

<!-- domain: work -->

> ⚠ **Piège** : le même sens français peut se construire différemment d'une langue à l'autre. Fiez-vous toujours à la **construction française du verbe**, pas à la traduction.

### 3.2 *le / la / les* (COD) vs *lui / leur* (COI) à la 3ᵉ personne

> *Tu regardes **la vidéo** ?* → *Tu **la** regardes ?* — *regarder qqch* = COD.
> *Tu réponds **au courriel** ?* → *Tu **y** réponds ?* — *répondre à qqch* = COI (chose → *y*, voir `gram-b1-08`).
> *Tu réponds **à tes amis** ?* → *Tu **leur** réponds ?* — *répondre à qqn* = COI (personne → *leur*).

**Règle** : pour une **personne** au COI, on emploie *lui* (singulier) ou *leur* (pluriel) ; pour une **chose** au COI, on emploie *y*.

> ⚠ **Piège** : *leur* pronom complément est **invariable** (jamais de *-s*). Ne pas le confondre avec le déterminant possessif *leurs* (*leurs amis*), lui variable. *Je **leur** parle* (jamais \*« je leurs parle »).

### 3.3 Les verbes-pièges : COI là où on attendrait un COD

Certains verbes très fréquents se construisent avec *à qqn* et exigent donc *lui / leur*, alors que l'intuition pousse vers *le / la / les* :

| Verbe (COI : *à qqn*) | Forme attendue | Forme fautive |
|---|---|---|
| téléphoner à qqn | *Je **lui** téléphone.* | \*je le téléphone |
| répondre à qqn | *Je **leur** réponds.* | \*je les réponds |
| obéir à qqn | *Les enfants **lui** obéissent.* | \*les enfants l'obéissent |
| plaire à qqn | *Ce film **leur** plaît.* | \*ce film les plaît |
| ressembler à qqn | *Elle **lui** ressemble.* | \*elle la ressemble |
| sourire à qqn | *Il **lui** sourit.* | \*il le sourit |

> *Le directeur **a écrit à ses employés**.* → *Le directeur **leur** a écrit.* — *écrire à qqn* = COI.
> *Le directeur **a convoqué ses employés**.* → *Le directeur **les** a convoqués.* — *convoquer qqn* = COD.

<!-- domain: politics -->

> ⚠ **Piège central du B1** : *aider*, *regarder*, *écouter*, *attendre*, *croire*, *connaître* se construisent **sans préposition** (COD → *le/la/les*), contrairement à *téléphoner / obéir / répondre / plaire* (COI → *lui/leur*). *Je l'écoute* (= *écouter qqn*), mais *je lui téléphone* (= *téléphoner à qqn*).

### 3.4 Position avec un verbe à un temps simple vs avec un infinitif

> *Je **les** achète au marché.* — temps simple : le pronom précède le verbe conjugué.
> *Je veux **les** acheter au marché.* — avec infinitif : le pronom précède **l'infinitif**, dont il est le complément.

**Règle** : le pronom se place **devant le verbe dont il est complément**. Avec *vouloir / pouvoir / devoir / aller / venir + infinitif*, c'est l'infinitif qui régit le pronom.

<!-- domain: daily-life -->

> ⚠ **Piège** : ne pas placer le pronom devant le verbe conjugué quand il complète l'infinitif. On dit *je veux **le** voir* (et non \*« je le veux voir »). De même au passé récent : *Je viens de **le** rencontrer.*

### 3.5 Place aux temps composés (avant l'auxiliaire)

> *J'**ai** rencontré Paul hier.* → *Je **l'ai** rencontré hier.* — le pronom précède **l'auxiliaire**, pas le participe.
> *Nous **avons** parlé à nos voisins.* → *Nous **leur avons** parlé.* — COI devant l'auxiliaire.

**Règle** : aux temps composés (passé composé, plus-que-parfait…), le pronom complément se place **devant l'auxiliaire** *avoir* ou *être*.

> ⚠ **Piège d'accord** : quand le COD *le / la / les* (ou le pronom relatif *que*) précède un participe passé conjugué avec *avoir*, le participe **s'accorde** avec ce COD antéposé.
> *Les fleurs que j'ai **achetées** sont fanées.* (*que* = *les fleurs*, fém. plur. → *-es*).
> *Cette lettre, je l'ai **reçue** ce matin.* (*l'* = *la lettre*, fém. sing. → *-e*).
> En revanche, **pas d'accord avec un COI** : *Je leur ai **parlé**.* (jamais \*« parlés »). Détail complet dans `gram-b1-13`.

### 3.6 Impératif affirmatif (pronom après le verbe) vs impératif négatif (pronom avant)

> *Regarde **le** tableau.* → **affirmatif** : *Regarde-**le**.* — le pronom suit le verbe, avec trait d'union.
> *Ne regarde pas **le** tableau.* → **négatif** : *Ne **le** regarde pas.* — le pronom reprend sa place habituelle, devant le verbe.

**Règle** : à l'impératif **affirmatif**, le ou les pronoms se placent **après** le verbe, reliés par des traits d'union. À l'impératif **négatif**, on revient à l'ordre normal (pronom devant le verbe).

> ⚠ **Piège des formes *moi / toi*** : à l'impératif affirmatif, *me* et *te* deviennent ***moi*** et ***toi***.
> *Donne-**moi** le livre.* (et non \*« donne-me »).
> *Lève-**toi**.*
> Mais à l'impératif **négatif**, on retrouve *me / te* devant le verbe : *Ne me donne pas ce livre.* / *Ne te lève pas.*

## 4. Faux amis et confusions à éviter

### 4.1 *lui* COI ≠ *le / la* COD

L'erreur la plus fréquente du B1 vient des verbes à complément indirect. Le critère est la **construction du verbe**, jamais le sens global :

| Verbe | Construction | Pronom 3ᵉ pers. | Exemple bien formé |
|---|---|---|---|
| connaître qqn | direct (COD) | le / la / les | *Je **la** connais.* |
| téléphoner à qqn | indirecte (COI) | lui / leur | *Je **lui** téléphone.* |
| écouter qqn | direct (COD) | le / la / les | *Je **l'**écoute.* |
| parler à qqn | indirecte (COI) | lui / leur | *Je **lui** parle.* |
| aider qqn | direct (COD) | le / la / les | *Je **l'**aide.* |
| obéir à qqn | indirecte (COI) | lui / leur | *Je **lui** obéis.* |

### 4.2 *leur* pronom (invariable) ≠ *leurs* déterminant (variable)

> *Je **leur** envoie un courriel.* — pronom COI, **toujours sans *-s***.
> *J'envoie un courriel à **leurs** collègues.* — déterminant possessif, accordé avec *collègues*.

Test : si le mot peut être remplacé par *lui* (singulier), c'est le pronom *leur* invariable. S'il accompagne un nom et marque la possession, c'est *leurs* (variable).

### 4.3 Erreurs d'ordre dans la double pronominalisation

> ✓ *Je **te le** présente.* (colonne 1 *te* + colonne 2 *le*) — et non \*« je le te présente ».
> ✓ *Je **le lui** ai prêté.* (colonne 2 *le* + colonne 3 *lui*) — et non \*« je lui le ai prêté ».
> ✓ *Il **nous en** a offert.* (colonne 1 *nous* + colonne 5 *en*) — et non \*« il en nous a offert ».

La grille du §2.2 est l'unique référence : *me/te/se/nous/vous* < *le/la/les* < *lui/leur* < *y* < *en*.

### 4.4 Verbes à double complément : l'objet de personne peut être COD ou COI selon le verbe

> *présenter qqn **à** qqn* : *Je **le lui** présente.* (le COD = la personne présentée ; le COI = celle à qui on présente).
> *aider qqn **à** faire qqch* : ici la personne aidée reste **COD** (*je l'aide à finir*), car *aider* se construit *aider qqn*.

> ⚠ Ne pas généraliser : la présence d'un *à* dans la phrase ne fait pas automatiquement de **tout** complément un COI. Il faut repérer **quel** complément est introduit par *à*.

## 6. Exercices

### 6.1 Repérage — 10 items (closed)

Pour chaque phrase, indiquez si le complément souligné est **COD** ou **COI**, et donnez le pronom qui le remplacerait (3ᵉ personne). Corrigé en §7.1.

1. J'écoute *la radio* tous les matins.
2. Je téléphone *à ma sœur* le dimanche.
3. Nous aidons *nos voisins* à déménager.
4. Le professeur répond *aux étudiants*.
5. Elle ressemble beaucoup *à sa mère*.
6. Tu connais *ce restaurant* ?
7. Ce projet plaît *au directeur*.
8. Les enfants obéissent *à leurs parents*.
9. J'ai invité *mes collègues* à la fête.
10. Il regarde *les photos* avec attention.

### 6.2 MCQ — 4 items (closed)

**Q1.** « Tu as parlé à Sophie ? — Oui, je _____ ai parlé hier. »
(a) la   (b) lui   (c) l'   (d) y

**Q2.** « Voici les documents ; je _____ ai envoyés ce matin. » *(à tes collègues)*
(a) leur les   (b) les leur   (c) leur l'   (d) lui les

**Q3.** « Ce livre est passionnant : il faut absolument _____ lire. »
(a) le lire   (b) lui lire   (c) y lire   (d) en lire

**Q4.** « Ne _____ donne pas ton numéro. » *(à cet inconnu)*
(a) le   (b) lui   (c) donne-lui   (d) leur

### 6.3 Transformation — 4 items

Récrivez la phrase en remplaçant les compléments soulignés par les pronoms qui conviennent. Corrigé en §7.3.

1. *J'ai prêté **ma voiture** **à mon frère**.*
2. *Le guide a montré **le musée** **aux touristes**.*
3. *N'oublie pas d'envoyer **le rapport** **à la directrice** !* (impératif négatif, puis affirmatif)
4. *Je veux offrir **ces fleurs** **à mes parents**.*

### 6.4 Production guidée — 2 items

Rédigez de courtes phrases (consignes ci-dessous). Modèles de correction en §7.4.

**P1.** En 4 ou 5 phrases, racontez une journée de travail en employant au moins **trois pronoms compléments différents** (un COD, un COI, et une double pronominalisation du type *je le lui ai…*).

**P2.** Donnez **trois conseils** à un ami qui prépare le TCF, à l'impératif affirmatif, en utilisant chaque fois un pronom complément (ex. *Écoute-les, répète-le, donne-toi du temps…*).

## 7. Corrigés

### 7.1 Repérage

| # | Complément | Fonction | Pronom | Justification |
|---|---|---|---|---|
| 1 | la radio | COD | *la* | *écouter qqch* (construction directe). |
| 2 | à ma sœur | COI | *lui* | *téléphoner à qqn*. |
| 3 | nos voisins | COD | *les* | *aider qqn* (direct, malgré le *à* de *à déménager*). |
| 4 | aux étudiants | COI | *leur* | *répondre à qqn*. |
| 5 | à sa mère | COI | *lui* | *ressembler à qqn*. |
| 6 | ce restaurant | COD | *le* | *connaître qqch*. |
| 7 | au directeur | COI | *lui* | *plaire à qqn*. |
| 8 | à leurs parents | COI | *leur* | *obéir à qqn*. |
| 9 | mes collègues | COD | *les* | *inviter qqn* (direct). |
| 10 | les photos | COD | *les* | *regarder qqch*. |

> Item 3 : le *à* appartient à *aider à + infinitif*, pas au complément de personne ; *aider* reste *aider qqn* → COD.

### 7.2 MCQ

- Q1 → **(b) lui** — *parler à qqn* = COI ; *l'* (élision de *le/la*) serait un COD, incorrect ici.
- Q2 → **(b) les leur** — COD *les* (colonne 2) avant COI *leur* (colonne 3). *(les leur ai envoyés : accord du participe avec le COD antéposé)*.
- Q3 → **(a) le lire** — *lire qqch* = COD ; le pronom précède l'infinitif. *(d)* emploie *en* (de + qqch), qui ne convient pas pour reprendre *ce livre*.
- Q4 → **(b) lui** — *donner à qqn* = COI ; à l'impératif **négatif**, le pronom reste **devant** le verbe.

### 7.3 Transformation

1. *Je **la lui** ai prêtée.* — COD *la* (*la voiture*, fém.) + COI *lui* ; accord du participe avec le COD antéposé (*prêtée*).
   <!-- AUDIT: accord du PP détaillé dans gram-b1-13 ; illustration minimale ici. -->
2. *Le guide **le leur** a montré.* — COD *le* (*le musée*) + COI *leur*.
3. *Négatif : Ne **le lui** envoie pas !* — ordre normal, pronoms devant le verbe.
   *Affirmatif : Envoie-**le**-**lui** !* — pronoms après le verbe, reliés par traits d'union, même ordre COD + COI.
4. *Je veux **les leur** offrir.* — COD *les* + COI *leur* ; les pronoms précèdent l'infinitif *offrir*.

### 7.4 Production — modèles de correction

## 8. Carte de synthèse

**COD vs COI — l'essentiel**

| Critère | COD | COI |
|---|---|---|
| Construction | *verbe + qqch/qqn* (sans préposition) | *verbe + à qqn* |
| Formes 1/2 pers. | me, te, nous, vous | me, te, nous, vous |
| Formes 3ᵉ pers. | **le / la / les** | **lui / leur** |
| Accord du PP (avec *avoir*) | oui, si COD **antéposé** | non |

**Verbes-pièges à COI (*lui / leur*)** : téléphoner à, répondre à, parler à, écrire à, obéir à, plaire à, ressembler à, sourire à.
**Verbes à COD (*le / la / les*)** : aider, regarder, écouter, attendre, croire, connaître, inviter, convoquer.

**Grille d'ordre (pronoms avant le verbe)**

| 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|
| me / te / se / nous / vous | le / la / les | lui / leur | y | en |

**Position selon la structure**

| Structure | Place du pronom | Exemple |
|---|---|---|
| Temps simple | devant le verbe | *Je **le** vois.* |
| Temps composé | devant l'auxiliaire | *Je **l'**ai vu.* |
| Verbe + infinitif | devant l'infinitif | *Je veux **le** voir.* |
| Impératif affirmatif | après le verbe (traits d'union ; *me → moi*) | *Donne-**le-moi**.* |
| Impératif négatif | devant le verbe (ordre normal) | *Ne **me le** donne pas.* |
