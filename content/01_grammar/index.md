# Grammaire

Référentiel gradué B1 → C2 organisé en quatre dossiers :

| Dossier | Cible | Phase de livraison |
|---|---|---|
| `b1_consolidation/` | Combler les lacunes B1 récurrentes (accords, temps composés, pronoms). | Phase 2 |
| `b2_core/` | Le palier le plus rentable au TCF : subjonctif, hypothèse, concession, discours rapporté, voix passive. | Phase 2 |
| `c1_advanced/` | Nominalisation, participe présent vs gérondif, articulation logique complexe. | Phase 2 |
| `c2_polish/` | Subjonctif imparfait passif, soutenu, nuances aspectuelles. | Phase 2 |

Chaque fiche : règle → exemples gradués → exercices → corrigé → 3 cartes Anki (i+1).
