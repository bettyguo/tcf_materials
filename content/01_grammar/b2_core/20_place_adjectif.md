# Place de l'adjectif — sémantique (avant vs. après le nom)

## 1. La question, en une phrase

> En français, **la majorité des adjectifs qualificatifs se placent après le nom** (position non marquée), et une petite classe fermée d'adjectifs courts et fréquents se place avant ; pour une dizaine d'entre eux, le **changement de position entraîne un changement de sens** (Riegel et al. 2018, ch. VI ; Grevisse & Goosse 2016, §§ sur l'épithète liée).
<!-- AUDIT: référence Riegel ch. VI (« le groupe nominal — l'épithète ») à confirmer selon l'édition ; ajuster avant clearance. -->

Conséquence pratique pour le TCF : il ne suffit pas de connaître la liste des adjectifs antéposables. Il faut savoir **lire la position comme un indice sémantique** (*un grand homme* ≠ *un homme grand*) et produire la position attendue selon le sens visé. C'est l'un des rares points où une erreur de syntaxe à l'écrit produit un contresens, pas seulement une maladresse.

## 2. Le principe général + les deux listes

**Règle par défaut** : l'adjectif épithète se place **après** le nom. C'est le cas pour la quasi-totalité des adjectifs : descriptifs, classifiants, dérivés, longs, à complément.

> *un livre intéressant / une maison rouge / une décision politique / une voiture neuve / un texte considérable / un projet fier de ses ambitions*

**Exception** : une classe fermée d'adjectifs courts et fréquents se place **avant** le nom (position antéposée non marquée).

### Tableau récapitulatif des positions

| Type d'adjectif | Position | Exemples |
|---|---|---|
| Adjectifs courts, fréquents, dits « subjectifs » | **Antéposés** par défaut | *beau, joli, jeune, vieux, nouveau, bon, mauvais, meilleur, pire, grand, petit, gros, vrai, faux, long, court, large, gentil, double, demi, autre, même* |
| Couleurs | **Postposés** (obligatoire) | *un mur blanc, une robe rouge, un ciel bleu* |
| Formes | **Postposés** (obligatoire) | *une table ronde, une boîte carrée, une cour rectangulaire* |
| Nationalités, religions, catégories classifiantes | **Postposés** (obligatoire) | *un écrivain français, une église catholique, une politique économique* |
| Participes employés comme adjectifs | **Postposés** (obligatoire) | *un enfant fatigué, un livre intéressant, un fait surprenant* |
| Adjectifs longs (≥ 3 syllabes) | **Postposés** | *un succès considérable, un projet impressionnant, un cas exceptionnel* |
| Adjectifs avec complément (à, de, en…) | **Postposés** (obligatoire) | *un livre plein d'erreurs, une personne fière de ses enfants, un homme bon à marier* |

> **Piège de mémorisation** : *neuf / neuve* est postposé (*une voiture neuve* = jamais utilisée), mais *nouveau / nouvelle* est antéposé (*une nouvelle voiture* = une autre, supplémentaire). Les deux ne sont pas synonymes : voir §3.3 et §4.

## 3. Les six sous-points contrastifs

### 3.1 La liste fermée des adjectifs antéposés

Une vingtaine d'adjectifs forment la classe des antéposés par défaut. Tous partagent trois propriétés : (i) courts (1–2 syllabes), (ii) fréquents, (iii) sémantiquement **évaluatifs ou quantifiants** plutôt que descriptifs (Riegel et al. 2018, ch. VI).

> *un **beau** paysage / une **jolie** maison / un **jeune** homme / un **vieux** quartier / une **nouvelle** voiture / un **bon** restaurant / une **mauvaise** idée / le **meilleur** choix / la **pire** solution / un **grand** projet / un **petit** problème / un **gros** effort / un **vrai** ami / un **faux** témoignage / un **long** voyage / un **court** délai / une **large** participation / un **gentil** garçon / une **double** vérification / un **autre** jour / la **même** question*

> ⚠ Cette liste n'est pas un absolu : tous ces adjectifs peuvent se postposer dans certains emplois (notamment quand ils sont eux-mêmes modifiés : *un homme grand et fort, un voyage long et coûteux*) — la postposition redonne alors une lecture descriptive (cf. §3.3).

### 3.2 Les adjectifs forcément postposés

Cinq catégories d'adjectifs **ne peuvent jamais** s'antéposer en français standard. La postposition est obligatoire.

| Catégorie | Test | Exemples |
|---|---|---|
| Couleurs | un seul terme objectif | *un drapeau bleu, blanc, rouge* — ✘ *un bleu drapeau* |
| Formes géométriques | propriété physique | *une table ronde, une boîte carrée* — ✘ *une ronde table* |
| Nationalités / origines | catégorie classifiante | *un philosophe allemand, un vin chilien* — ✘ *un allemand philosophe* |
| Participes adjectivés | dérivés verbaux | *un enfant fatigué, un projet abouti, un fait surprenant* — ✘ *un fatigué enfant* |
| Adjectifs longs (≥ 3 syll.) | poids prosodique | *un succès considérable, un projet impressionnant* — ✘ *un considérable succès* |

> ⚠ La règle des adjectifs longs n'est pas absolue mais statistique : *un magnifique paysage* est attesté à l'écrit littéraire (antéposition stylistique, voir `gram-c1-14`), mais à l'écrit standard et au TCF, la postposition reste la norme. Grevisse & Goosse (2016) notent que l'antéposition d'adjectifs longs est un effet de style, pas une option neutre.

### 3.3 **Les changements de sens selon la position** (le piège central)

Pour une dizaine d'adjectifs, la position devant ou derrière le nom **change le sens** : antéposé = lecture **figurée, subjective, évaluative** ; postposé = lecture **propre, objective, descriptive** (Riegel et al. 2018, ch. VI ; Grevisse & Goosse 2016).

Voici les **douze paires à mémoriser** :

| Adjectif | Antéposé (sens figuré / subjectif) | Postposé (sens propre / objectif) |
|---|---|---|
| **grand** | *un **grand** homme* = illustre, important | *un homme **grand*** = de haute taille |
| **ancien** | *un **ancien** ministre* = qui n'est plus en poste | *un meuble **ancien*** = vieux, antique |
| **propre** | *son **propre** livre* = à lui-même (possession) | *un livre **propre*** = non sale |
| **pauvre** | *un **pauvre** homme* = à plaindre, malheureux | *un homme **pauvre*** = sans argent |
| **seul** | *une **seule** femme* = uniquement une | *une femme **seule*** = isolée, sans compagnie |
| **cher** | *un **cher** ami* = qui m'est cher (affection) | *un livre **cher*** = coûteux |
| **certain** | *un **certain** succès* = d'un certain niveau (indéfini) | *un succès **certain*** = assuré, garanti |
| **brave** | *un **brave** homme* = gentil, honnête | *un homme **brave*** = courageux |
| **différents** | ***différents** avis* = plusieurs, divers | *des avis **différents*** = qui diffèrent entre eux |
| **simples** | *de **simples** questions* = rien que des questions | *des questions **simples*** = faciles |
| **dernier** | *un **dernier** effort* = ultime, final | *l'an **dernier*** = le précédent (postposé par défaut pour les temps) |
| **drôle** | *une **drôle** d'histoire* = étrange (avec *de*) | *une histoire **drôle*** = qui fait rire |

> ⚠ **Piège central du TCF** : les paires *grand / ancien / propre / pauvre / cher / seul* sont les plus testées en compréhension écrite. Lire *un ancien collègue* comme « un collègue d'autrefois » et non « un collègue de grande ancienneté » est un réflexe à acquérir.

> ⚠ **Note sur *drôle*** : la construction antéposée exige la préposition *de* (*une drôle d'histoire, un drôle de type*) ; postposée, elle se construit directement (*une histoire drôle*). C'est l'un des rares cas où la position change aussi la syntaxe.
<!-- AUDIT: confirmer que la construction *un drôle de N* est analysée par Riegel comme antéposition sémantique (et non comme locution figée à part) ; ajuster la formulation avant clearance. -->

> ⚠ **Note sur *dernier*** : *un dernier effort* (ultime, antéposé) vs *l'an dernier, la semaine dernière* (le précédent, postposé). La règle « antéposé = figuré » fonctionne ici aussi : « ultime, final » est une évaluation ; « précédent dans le temps » est une localisation temporelle objective, donc postposée.

### 3.4 Coordination de plusieurs adjectifs

Trois configurations sont possibles selon la nature des adjectifs coordonnés.

**(a) Adjectifs coordonnés tous postposables** → tous après le nom, reliés par *et* :
> *un livre **intéressant et bien écrit*** / *une décision **politique et économique***

**(b) Deux adjectifs tous deux antéposables** → cumul possible avant le nom :
> *un **beau grand** jardin* / *une **jeune jolie** femme* / *un **vieux bon** ami*

**(c) Un antéposable + un postposable** → de part et d'autre du nom :
> *une **jeune** femme **française*** / *un **petit** garçon **brun*** / *un **vieux** quartier **populaire***

> ⚠ La configuration (b) est moins fréquente à l'écrit standard ; on préfère souvent **antéposé + nom + coordonné postposé** : *une jeune femme intelligente et travailleuse*. À l'oral et en littérature, le cumul antéposé est attesté (*un beau grand jardin*) mais sonne marqué.

### 3.5 *De* devant un adjectif antéposé pluriel

Quand un adjectif **antéposé pluriel** précède un nom au pluriel, l'article indéfini *des* devient **de** :

> *J'ai planté **de** beaux arbres dans le jardin.*
> *Elle a eu **de** bonnes idées.*
> *Ce sont **de** vrais amis.*

Comparer :
> *J'ai planté **des** arbres magnifiques.* (adjectif postposé → *des* maintenu)
> *Ce sont **des** amis vrais et fidèles.* (adjectif postposé → *des* maintenu)

> ⚠ Règle de la grammaire normative codifiée par Grevisse & Goosse (2016). À l'oral courant, *des beaux jardins* s'entend largement ; à l'écrit TCF (registre formel), garder **de**. Voir `gram-b1-01` §3.4 pour la règle générale du partitif et le tableau complet.

### 3.6 Adjectifs avec complément — toujours postposés

Dès qu'un adjectif est suivi d'un **complément** introduit par une préposition (*à, de, en, par…*), il est obligatoirement postposé, même s'il appartient normalement à la classe des antéposables.

> *un **bon** restaurant* (antéposé) → *un restaurant **bon à recommander*** (postposé car complément)
> *un **grand** projet* (antéposé) → *un projet **grand par son ambition*** (postposé)
> *une **belle** femme* (antéposé) → *une femme **belle de toute sa jeunesse*** (postposé)

Exemples supplémentaires :
> *un livre **plein** d'erreurs / une mère **fière** de son fils / un travail **utile** à la communauté / un argument **difficile** à réfuter*

> ⚠ Cette règle prévaut sur la classe lexicale : *bon, beau, grand* basculent automatiquement en position postposée dès qu'ils prennent un complément. C'est une **règle de syntaxe**, pas de style.

## 4. Faux amis et confusions à éviter

### 4.1 *Neuf* vs *nouveau*

Les deux se traduisent parfois par *new* mais ne sont pas synonymes :

| Adjectif | Position | Sens | Exemple |
|---|---|---|---|
| **nouveau / nouvelle** | antéposé | autre, supplémentaire, différent du précédent | *une **nouvelle** voiture* (= je change de voiture ; elle peut être d'occasion) |
| **neuf / neuve** | postposé | jamais utilisé, sortant de production | *une voiture **neuve*** (= sortie d'usine, jamais roulée) |

> ⚠ *J'ai acheté une nouvelle voiture* ne dit rien sur l'état du véhicule ; *j'ai acheté une voiture neuve* précise qu'elle n'a pas servi.

### 4.2 *Propre* — trois sens selon la position et le déterminant

| Construction | Sens | Exemple |
|---|---|---|
| Possessif + ***propre*** + N | renforcement de la possession (à lui-/elle-même) | *son **propre** livre* (= le sien, pas celui d'un autre) |
| N + ***propre*** | non sale | *un drap **propre*** |
| ***propre** à* + N/inf. | spécifique à, caractéristique de | *un trait **propre** à cette culture* |

### 4.3 *Certain* — antéposé indéfini vs postposé assuré

> *J'ai eu un **certain** succès dans cette démarche.* (un certain niveau, modéré, indéfini)
> *Mon succès est **certain**.* (assuré, garanti — postposé en position attribut, mais aussi épithète : *un succès **certain***)

L'antéposé *un certain N* fonctionne comme un quasi-déterminant indéfini ; le postposé *un N certain* est un adjectif évaluatif au sens plein.

### 4.4 Anglicismes de position à éviter

L'anglais antépose la plupart de ses adjectifs ; le français ne le fait pas. Les calques suivants sont fautifs :

| ✘ Anglicisme | ✓ Français |
|---|---|
| *une rouge robe* (calque de *a red dress*) | *une robe rouge* |
| *un français écrivain* (calque de *a French writer*) | *un écrivain français* |
| *un considérable effort* (calque de *a considerable effort*) | *un effort considérable* |
| *un fatigué enfant* (calque de *a tired child*) | *un enfant fatigué* |

## 7. Corrigés

### 7.1 Repérage

| # | Syntagme | Position | Justification |
|---|---|---|---|
| 1 | un livre intéressant | **P** | participe adjectivé → toujours postposé. |
| 2 | une grande maison | **A** | *grand* appartient à la liste fermée des antéposés. |
| 3 | une décision politique | **P** | adjectif classifiant (catégorie) → postposé. |
| 4 | de beaux jardins | **A** | *beau* antéposé + règle *des* → *de* devant adj. antéposé pluriel. |
| 5 | un meuble ancien | **P** | sens propre (vieux) → postposition ; antéposé donnerait « un ancien meuble » (qui n'est plus un meuble). |
| 6 | un projet considérable | **P** | adjectif long (≥ 3 syllabes) → postposé. |
| 7 | un bon restaurant | **A** | *bon* appartient à la liste fermée. |
| 8 | une femme fière de ses enfants | **P** | adjectif avec complément (*de ses enfants*) → postposé obligatoire. |

### 7.2 Choix de sens

- Q1 → **(b)** un homme illustre. *grand* antéposé = figuré/évaluatif.
- Q2 → **(b)** un meuble vieux, d'époque. *ancien* postposé = sens propre (âge).
- Q3 → **(a)** son livre à lui-même. *propre* antéposé après possessif = renforcement de possession.
- Q4 → **(a)** un homme à plaindre. *pauvre* antéposé = figuré (sentiment de pitié).
- Q5 → **(a)** uniquement une femme. *seul* antéposé = quantifiant exclusif.
- Q6 → **(b)** un ami qui m'est cher. *cher* antéposé = sens affectif.
- Q7 → **(b)** un succès d'un certain niveau, modéré. *certain* antéposé = indéfini évaluatif.
- Q8 → **(a)** plusieurs avis. *différents* antéposé = quantifiant indéfini (= *plusieurs, divers*).

### 7.3 Transformation

1. *un effort **considérable*** (adjectif long → postposition).
2. ***de** nouvelles idées* (*nouveau* antéposable + pluriel → *des* → *de*).
3. *une voiture **rouge*** (couleur → postposition obligatoire).
4. *un **grand** homme* (sens « illustre » → antéposition).

### 7.4 Correction d'erreurs

1. ✓ *un **écrivain français*** — la nationalité est postposée en français.
2. ✓ *un **projet considérable*** — adjectif long, postposé.
3. ✓ *un **voyageur fatigué*** — participe adjectivé, postposé obligatoire.
4. ✓ *de beaux paysages* — *des* devient *de* devant un adjectif antéposé pluriel (Grevisse & Goosse 2016 ; voir `gram-b1-01` §3.4).

### 7.5 Choix sémantique

1. *un homme **pauvre*** (postposé : sans argent, sens propre).
2. *un **pauvre** homme* (antéposé : à plaindre, sens figuré).
3. *une femme **seule*** (postposé : isolée, sans compagnie).
4. *une **seule** femme* (antéposé : uniquement une, quantifiant).

**Grilles d'évaluation succinctes** (à utiliser en auto-correction) :

| Critère | Acceptable si… |
|---|---|
| Position juste | chaque adjectif est placé selon sa classe lexicale et le sens visé. |
| Sens contrôlé | aucun contresens involontaire dû à une mauvaise position (cf. §3.3). |
| Règle *des* → *de* | respectée devant adjectif antéposé pluriel à l'écrit. |
| Pas d'anglicisme | aucune antéposition d'adjectif normalement postposé (couleur, nationalité, long). |
| Cohérence du registre | écrit formel, aucun anglicisme (cf. `tools/anglicisms.yaml`). |

## 8. Carte de synthèse

> **(Cette section est destinée à être extraite par le futur build des cheatsheets — voir spec §6. Garder concise.)**

**Règle par défaut** : adjectif **après** le nom. Exception : classe fermée d'adjectifs courts et fréquents **avant** le nom.

**Liste des antéposés par défaut** (à mémoriser) : *beau, joli, jeune, vieux, nouveau, bon, mauvais, meilleur, pire, grand, petit, gros, vrai, faux, long, court, large, gentil, double, demi, autre, même*.

**Toujours postposés (obligatoire)** : couleurs · formes · nationalités · participes adjectivés · adjectifs longs · adjectifs avec complément.

**Tableau des paires à sens variable (mémoriser en couples)** :

| Antéposé (figuré) | Postposé (propre) |
|---|---|
| un **grand** homme (illustre) | un homme **grand** (de taille) |
| un **ancien** ministre (ex-) | un meuble **ancien** (vieux) |
| son **propre** livre (à lui) | un livre **propre** (non sale) |
| un **pauvre** homme (à plaindre) | un homme **pauvre** (sans argent) |
| une **seule** femme (uniquement) | une femme **seule** (isolée) |
| un **cher** ami (affectif) | un livre **cher** (coûteux) |
| un **certain** succès (indéfini) | un succès **certain** (assuré) |
| un **brave** homme (gentil) | un homme **brave** (courageux) |
| **différents** avis (plusieurs) | des avis **différents** (qui diffèrent) |
| de **simples** questions (rien que) | des questions **simples** (faciles) |
| un **dernier** effort (ultime) | l'an **dernier** (le précédent, temps) |
| une **drôle** d'histoire (étrange) | une histoire **drôle** (qui fait rire) |

**Mnémo central** : **antéposé = subjectif / figuré / évaluatif ; postposé = objectif / propre / descriptif**.

**Règle des partitifs** : *des* → **de** devant un adjectif antéposé pluriel (*de beaux jardins*).
