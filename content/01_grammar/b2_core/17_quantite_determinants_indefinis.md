# Expression de la quantité — déterminants indéfinis (plupart, quelques, certains, divers, plusieurs)

## 1. La question, en une phrase

> Les déterminants indéfinis de quantité actualisent un nom en indiquant une **quantité non chiffrée** ; chacun impose sa propre contrainte d'accord (nombre, genre, accord du verbe), et l'erreur ne porte presque jamais sur le sens mais sur la morphologie (Riegel et al. 2018, ch. VI).
<!-- AUDIT: référence Riegel ch. VI (le groupe nominal / les déterminants) à confirmer — chapitre exact à vérifier selon l'édition. -->

Conséquence pratique pour le TCF : chaque déterminant a un comportement *non-prédictible à partir du sens*. *La plupart* est singulier mais commande le pluriel ; *plusieurs* est invariable en genre ; *chaque* refuse le pluriel ; *différents* change de sens selon sa position. Le test interroge précisément ces points.

## 2. Les principaux indéfinis de quantité — vue d'ensemble

| Déterminant | Accord | Emploi typique |
|---|---|---|
| *la plupart (des / de + GN défini)* | sing. en forme ; **verbe au pluriel** si *la plupart des* + nom plur. | quantité majoritaire : *la plupart des Français* |
| *quelques* | invariable, **pluriel** seulement | petit nombre comptable : *quelques minutes* |
| *un peu de* | invariable, **singulier** non comptable | petite quantité de matière : *un peu de patience* |
| *certains, certaines* | accord en **genre**, pluriel | sous-ensemble : *certaines régions* |
| *plusieurs* | **invariable** (ni genre, ni singulier) | plus de deux : *plusieurs hypothèses* |
| *divers, diverses ; différents, différentes* | accord genre + nombre ; **antéposé** = "plusieurs" | *divers facteurs, différentes solutions* |
| *chaque* | toujours **singulier** ; pronom : *chacun(e)* | distributif : *chaque candidat* |
| *tout, toute, tous, toutes* | accord en genre et nombre | totalité : *tous les jours* |
| *aucun, aucune* | **singulier** (sauf cas spéciaux) ; négatif | *aucun problème, aucune réponse* |
| *n'importe quel(le), quelconque* | accord en genre/nombre | indifférenciation : *n'importe quel jour, un jour quelconque* |

> ⚠ Aucun de ces déterminants ne se combine avec un article (*le, la, les, un, une, des*) : on dit *plusieurs livres*, pas ✘ *les plusieurs livres* ; *chaque jour*, pas ✘ *un chaque jour*. Exception : *la plupart* contient déjà l'article *la*.

## 3. Étude contrastive, sous-section par sous-section

### 3.1 *La plupart (de)* — accord du verbe au pluriel

Règle : après *la plupart des* + nom **pluriel**, le verbe et l'attribut s'accordent au **pluriel**, jamais au singulier (Grevisse & Goosse 2016, *Accord du verbe avec un collectif*).
<!-- AUDIT: référence Grevisse §428 / §437 (accord avec collectifs) à confirmer — paragraphe exact selon l'édition 2016. -->

> ✓ *La plupart des étudiants **sont** satisfaits du cours.*
> ✘ *La plupart des étudiants **est** satisfaite du cours.* — faute classique, sanctionnée au TCF.

Le sujet réel, pour l'accord, est le **complément** *des étudiants*, et non le mot *plupart* (même mécanisme que *une foule de gens **arrivent***).

**Variante avec un singulier de matière** — *la plupart de + GN défini singulier* :

> ✓ *La plupart de mon temps **est** consacrée à la recherche.* (verbe **singulier**, complément singulier)
> ✓ *La plupart du public **a** apprécié.* (singulier)

**Sans complément** (anaphorique) : pluriel par défaut.

> ✓ *La plupart **pensent** que la réforme arrivera trop tard.*

**Comparaison de structure** — autres expressions de quantité majoritaire qui imposent l'accord au pluriel :

| Tournure | Accord du verbe |
|---|---|
| *bien des* + plur. | pluriel : *Bien des candidats **ont** échoué.* |
| *nombre de* + plur. | pluriel : *Nombre de chercheurs **estiment**…* |
| *beaucoup de* + plur. | pluriel : *Beaucoup d'électeurs **votent** blanc.* |
| *la majorité des* + plur. | usuel pluriel ; singulier possible si on insiste sur le bloc unitaire (Grevisse 2016). |

> ⚠ **Piège de mémoire** : ne pas raisonner par analogie avec *un grand nombre* (qui admet l'accord au singulier) — pour *la plupart*, l'usage moderne est fermement **pluriel**.

### 3.2 *Quelques* vs *un peu de* — comptable vs non comptable

*Quelques* sélectionne un nom **comptable pluriel** ; *un peu de* sélectionne un nom **non comptable singulier**. Les deux signifient "petite quantité", mais leurs domaines d'emploi sont disjoints (Riegel et al. 2018).

| Comptable (*quelques* + pluriel) | Non comptable (*un peu de* + singulier) |
|---|---|
| *quelques amis* | *un peu d'amitié* |
| *quelques minutes* | *un peu de temps* |
| *quelques erreurs* | *un peu de patience* |
| *quelques arguments* | *un peu de bon sens* |

> ✘ *J'ai un peu d'amis.* (calque possible de l'anglais *a few of friends*)
> ✓ *J'ai **quelques** amis.*

> ✘ *Donne-moi quelques pain.*
> ✓ *Donne-moi **un peu de** pain.*

**Note de registre** : *quelques* (~ "un petit nombre") ne doit pas être confondu avec **quelque** (sans -s, adverbe ou déterminant indéfini singulier) :

- *Il a **quelque** difficulté à s'exprimer.* — déterminant singulier, registre soutenu.
- *Quelque **trois cents** personnes assistaient au débat.* — adverbe invariable devant un numéral (= environ).

> ⚠ Au TCF écrit, l'orthographe *quelque/quelques* est régulièrement testée : retenir que *quelques* prend -s si et seulement si le nom qui suit est au **pluriel**.

### 3.3 *Certains, certaines* — accord en genre, pronom autonome

Comme déterminant, *certains* s'accorde en **genre et en nombre** avec le nom qu'il détermine ; il n'a pas de forme singulière comme déterminant indéfini ("un certain" relève d'un autre emploi, voir ci-dessous).

> ✓ *Certains films me laissent indifférent.*
> ✓ *Certaines personnes refusent d'en parler.*

Comme **pronom**, *certains, certaines* fonctionne sans nom :

> ✓ ***Certains pensent** que la situation se redressera ; **d'autres** en doutent.*

**À ne pas confondre** :

| Forme | Sens | Exemple |
|---|---|---|
| *certains + nom plur.* | sous-ensemble ("quelques") | *certains experts contestent ce résultat* |
| *un certain + nom sing.* | indétermination + indéfinition | *un certain Monsieur Durand* (= qu'on ne connaît pas) |
| *une certaine + nom sing.* | nuance d'évaluation | *une certaine lassitude se fait sentir* |
| *certain* (adjectif postposé) | "sûr, indubitable" | *un fait **certain** ; une victoire **certaine*** |

> ⚠ *Un certain succès* ≠ *un succès certain* : antéposé = nuance / atténuation ; postposé = "qui est sûr".

### 3.4 *Plusieurs* — invariable, toujours pluriel

*Plusieurs* est **invariable en genre** (une seule forme pour le masculin et le féminin) et n'existe **qu'au pluriel**. Il signifie "plus de deux, plus d'un certain nombre" (Riegel et al. 2018).

> ✓ *plusieurs étudiants* / *plusieurs étudiantes* (même forme)
> ✓ *plusieurs questions / plusieurs livres*

Comme pronom :

> ✓ *Parmi les candidats, **plusieurs** ont retiré leur dossier.*

**Contrastes à mémoriser** :

| ✓ | ✘ |
|---|---|
| *plusieurs étudiantes* | ✘ *plusieures étudiantes* |
| *plusieurs reprises* | ✘ *plusieures reprises* |
| *plusieurs d'entre elles* | ✘ *plusieures d'entre elles* |

> ⚠ La graphie ✘ *plusieures* n'existe pas. C'est l'une des fautes de copie les plus régulières au TCF EE.

### 3.5 *Divers, différents* — antéposés vs postposés

Antéposés au nom, *divers* et *différents* (plus rarement *différentes, diverses*) signifient "**plusieurs, variés**" ; postposés, ils retrouvent leur sens d'adjectif qualificatif (Grevisse & Goosse 2016).

| Position | Sens | Exemple |
|---|---|---|
| **antéposé** | "plusieurs / divers" | *Différents auteurs ont traité cette question.* |
| **postposé** | "qui diffèrent, qui ne se ressemblent pas" | *Les auteurs ont des positions différentes.* |
| **antéposé** | "plusieurs / variés" | *Diverses raisons expliquent cet échec.* |
| **postposé** | "varié, hétérogène" | *Des paysages divers se succèdent.* |

> ✓ *Différents experts ont été consultés.* (= plusieurs experts, sans jugement sur leur degré de divergence)
> ✓ *Les experts consultés ont rendu des conclusions différentes.* (= leurs conclusions ne coïncident pas)

> ⚠ La position renverse le sens : ne pas écrire *différents avis* en pensant à des avis divergents — préférer *avis différents* dans ce cas.

**Cas particulier** : *différent* après un nom au singulier garde son sens plein.

> ✓ *Une approche différente est possible.* (= autre, qui se distingue)

### 3.6 *Chaque* — distributif, singulier obligatoire ; pronom *chacun(e)*

*Chaque* est **toujours singulier** : il distribue la propriété sur les éléments d'un ensemble, un par un (Riegel et al. 2018).

> ✓ *Chaque étudiant doit présenter un exposé.*
> ✓ *Chaque jour apporte son lot de surprises.*
> ✘ *Chaque étudiants…* / ✘ *chaques jours…*

Le **pronom correspondant** est *chacun* (masc.) / *chacune* (fém.) :

> ✓ ***Chacun** porte la responsabilité de ses choix.*
> ✓ ***Chacune** de ces solutions a ses limites.*

> ⚠ **Piège classique** : devant un numéral, on emploie *tous les* + numéral, non *chaque* + numéral.
> ✘ *Le bus passe chaque dix minutes.* (calque possible de l'anglais *every ten minutes*)
> ✓ *Le bus passe **tous les dix minutes**.* — règle confirmée par la BDL (OQLF), entrée *chaque*.

### 3.7 *Tout, toute, tous, toutes* — déterminant, pronom, adverbe

*Tout* est l'un des mots les plus polyfonctionnels du français. Trois emplois principaux à distinguer (Riegel et al. 2018).

**a) Déterminant** : s'accorde en genre et en nombre avec le nom.

> ✓ *tout le monde, toute la journée, tous les jours, toutes les fois*

**b) Pronom** : reprend un GN.

> ✓ *Les invités ? Ils sont **tous** venus.* (s prononcé : [tus])
> ✓ *Les solutions ? Je les ai **toutes** examinées.*

> ⚠ Le **-s final de *tous* pronom se prononce** ([tus]), alors qu'il est muet quand *tous* est déterminant (*tous les jours* [tu le ʒuʁ]).

**c) Adverbe** (devant adjectif ou adverbe, = "entièrement, tout à fait") : **invariable** sauf devant un adjectif féminin commençant par une **consonne ou un h aspiré**, où il s'accorde par euphonie (Grevisse & Goosse 2016).

| Contexte | Forme | Exemple |
|---|---|---|
| adj. masc. | invariable *tout* | *Il est **tout** étonné.* |
| adj. fém. commençant par voyelle ou h muet | invariable *tout* | ***Tout** étonnée, elle resta muette.* / ***Tout** heureuse de la nouvelle.* (h muet) |
| adj. fém. commençant par consonne ou h aspiré | accord : *toute, toutes* | ***Toute** heureuse* (h aspiré, rare) ; ***Toutes** contentes du résultat.* |

> ⚠ Cette règle de *tout* adverbe est l'un des items orthographiques les plus testés au B2 / C1. Mémoriser le couple :
> ✓ *tout étonnée* (voyelle → invariable)
> ✓ *toute surprise* (consonne → accord)

### 3.8 *Aucun, aucune* — singulier, négatif

*Aucun(e)* est un déterminant **négatif** ; il s'emploie au **singulier** et appelle *ne* dans la principale (Riegel et al. 2018).

> ✓ *Je n'ai **aucun** problème avec cette décision.*
> ✓ *Elle ne fait **aucune** erreur.*

**Cas spécial — noms toujours pluriels** : avec des mots qui n'existent qu'au pluriel (*frais, funérailles, vivres, mœurs, ténèbres*), *aucun* prend exceptionnellement la marque du pluriel (Grevisse & Goosse 2016).
<!-- AUDIT: confirmer la liste exacte des noms pluralia tantum tolérant "aucuns" — Grevisse cite un sous-ensemble restreint. -->

> ✓ ***Aucuns frais** ne seront facturés.*
> ✓ ***Aucunes funérailles** n'ont été annoncées.*

> ⚠ Hors de ce cas marginal, *aucun(e)* est **toujours singulier**. Ne pas écrire ✘ *aucuns problèmes, aucunes erreurs*.

**Pronom** : *aucun(e)* fonctionne aussi sans nom :

> ✓ *Parmi ses propositions, **aucune** ne m'a convaincu.*

### 3.9 *N'importe quel(le), quelconque* — indifférenciation

Ces deux tournures expriment l'**indifférence du choix**, mais leur position et leur registre diffèrent (Grevisse & Goosse 2016).

| Forme | Position | Registre | Exemple |
|---|---|---|---|
| *n'importe quel(le)* | **antéposé** au nom | courant | *Choisissez **n'importe quel** jour.* |
| *quelconque* | **postposé** au nom | plus neutre / soutenu | *Choisissez un jour **quelconque**.* |

> ✓ *Pour **n'importe quelle** raison, le système peut tomber en panne.*
> ✓ *Pour une raison **quelconque**, le système peut tomber en panne.*

> ⚠ Deux écueils : (1) *n'importe quel* s'accorde (*n'importe quelle, n'importe quels, n'importe quelles*) ; (2) *quelconque* placé **avant** un nom prend un sens péjoratif ("médiocre, sans intérêt") :
> *un argument **quelconque*** (= n'importe lequel) **≠** *un **quelconque** argument* (= un argument banal, sans valeur).

## 4. Faux amis et confusions à éviter

### 4.1 Singulier morphologique mais accord pluriel

| Tournure | Forme | Accord du verbe |
|---|---|---|
| *la plupart des* + plur. | sing. | **pluriel** |
| *bien des* + plur. | sing. | **pluriel** |
| *la majorité des* + plur. | sing. | pluriel usuel |
| *un grand nombre de* + plur. | sing. | pluriel ou singulier (sing. si on insiste sur l'unité) |
| *une foule de* + plur. | sing. | pluriel usuel |

> ⚠ La règle prototypique du français — "*sujet singulier → verbe singulier*" — **ne s'applique pas** à ces collectifs lorsqu'ils sont suivis d'un complément pluriel : c'est ce complément qui pilote l'accord (Grevisse & Goosse 2016, *Accord du verbe avec un collectif*).

### 4.2 Indéfinis qui n'ont pas de partitif

Les déterminants indéfinis de quantité **excluent l'article partitif** *du, de la, des* devant le nom qu'ils déterminent.

> ✘ *plusieurs des étudiants* (= "plusieurs parmi les étudiants" est correct comme pronom + complément, mais ≠ déterminant)
> ✓ ***plusieurs étudiants*** (déterminant)
> ✓ ***plusieurs d'entre les étudiants*** / ***plusieurs des étudiants présents*** (pronom partitif)

> ✘ *quelques des amis* → ✓ ***quelques amis***
> ✘ *certains des films* (comme déterminant) → ✓ ***certains films*** ; ✓ ***certains des films récents*** (pronom partitif)

### 4.3 Faux amis avec l'anglais

| Anglais | Calque erroné | Forme française |
|---|---|---|
| *most of the* | ✘ *la plus part des* | ✓ ***la plupart des*** (un seul mot, *plupart*) |
| *every five minutes* | ✘ *chaque cinq minutes* | ✓ ***tous les cinq minutes*** |
| *a few of my friends* | ✘ *un peu de mes amis* | ✓ ***quelques-uns de mes amis*** |
| *several persons* | ✘ *plusieures personnes* | ✓ ***plusieurs personnes*** (invariable) |
| *each of the* | ✘ *chaque des* | ✓ ***chacun(e) des*** |

> ⚠ Le calque *la plus part* (en trois mots) est une faute d'orthographe fréquente même chez les francophones — *plupart* est soudé.

### 4.4 Constructions équivalentes à éviter de confondre

- *quelques* ≠ *quelques-uns* : *quelques* est un **déterminant** (suivi du nom) ; *quelques-uns / quelques-unes* est le **pronom** (autonome, accord en genre).
  > *J'ai **quelques** questions.* (dét.) / *J'en ai **quelques-unes**.* (pron.)
- *chaque* ≠ *chacun(e)* : déterminant vs pronom, même rapport.
  > ***Chaque** participant reçoit une feuille.* / ***Chacun** reçoit une feuille.*
- *tout le monde* (singulier, pronom indéfini collectif) prend un **verbe au singulier** :
  > ✓ *Tout le monde **est** parti.* (jamais ✘ *sont*)

### 6.4 Production guidée — 2 items

Rédigez 2–3 phrases (40–60 mots) en utilisant les déterminants indiqués au moins une fois chacun. Modèle de correction en §7.4.

**P2.** Vous parlez de vos habitudes quotidiennes. Utilisez *chaque* (sing.) et *tous les* + numéral (ou expression temporelle) au moins une fois chacun, plus un emploi de *plusieurs*.

| 8 | **tous** | pronom masc. plur. (s prononcé) ; reprend *les invités*. |
| 9 | **Certains** | déterminant indéfini, accord avec *films* masc. plur. |
| 10 | **est** | *la plupart de mon temps* : complément singulier non comptable → verbe singulier. |

### 7.2 MCQ

- Q1 → **(b) sont** — *la plupart des participants* → verbe au pluriel.
- Q2 → **(c) quelques** — *erreurs* est un comptable pluriel.
- Q3 → **(a) Chaque** — distributif singulier antéposé au nom (b *Chacun* est un pronom et ne peut précéder un nom).
- Q4 → **(c) n'importe quelle** — accord avec *raison* fém. sing. ; *quelconque* serait possible mais en **postposition** (*pour une raison quelconque*).

### 7.3 Transformation

1. ***La plupart des** électeurs **n'iront pas** voter.* (verbe au pluriel ; *des* est obligatoire avec *la plupart*)
2. ***Chaque** jour, il prend le métro à la même heure.* (singulier obligatoire)
3. ***Aucun** n'a refusé la proposition.* (singulier ; négation avec *ne*) — variante : ***Personne** n'a refusé la proposition.*
4. *Il y a **plusieurs** solutions possibles, aux coûts **différents**.* (postposé = qui diffèrent) — ou : *Il y a **différentes** solutions possibles, qui se distinguent par leur coût.* (antéposé = "plusieurs"). Les deux sont acceptables ; le choix dépend du nuance visée.

### 7.4 Production — modèles de correction

> ⚠ Productions auctoriales en attente de revue native. Confidence : **medium**.

**P1 (modèle, domaine recherche / travail)** :
> La plupart des participants à notre étude **ont** déclaré une amélioration de leur qualité de vie après six mois. Certaines variables, en revanche, restent difficiles à interpréter : il est probable que nous devrons élargir l'échantillon pour conclure.

**P2 (modèle, domaine vie quotidienne)** :
> Chaque matin, je commence ma journée par une demi-heure de lecture. Je prends le métro tous les jours à la même heure et j'ai plusieurs collègues qui font le même trajet : nous nous croisons sans nous parler, perdus dans nos écouteurs.

**Grilles d'évaluation succinctes** (à utiliser en auto-correction) :

| Critère | Acceptable si… |
|---|---|
| Accord du verbe | *la plupart des* + plur. → verbe au pluriel ; *chaque* / *aucun* + sing. → verbe au singulier ; *tout le monde* → singulier. |
| Choix du déterminant | comptable plur. → *quelques* / *plusieurs* / *certains* ; non comptable sing. → *un peu de* ; distributif → *chaque*. |
| Position de *divers / différents* | antéposé = "plusieurs" ; postposé = "qui diffèrent" — cohérent avec le sens visé. |
| Cohérence du registre | écrit formel, aucun anglicisme (cf. `tools/anglicisms.yaml`) ; *présentement* exclu (Québec) → *actuellement*. |

## 8. Carte de synthèse

> **(Cette section est destinée à être extraite par le futur build des cheatsheets — voir spec §6. Garder concise.)**

**Déterminants indéfinis de quantité — règles d'accord clés**

| Déterminant | Singulier / Pluriel | Accord du verbe |
|---|---|---|
| *la plupart des* + plur. | sing. en forme | **pluriel** |
| *la plupart de* + GN sing. | sing. | singulier |
| *quelques* | pluriel comptable | (selon nom) |
| *un peu de* | sing. non comptable | singulier |
| *certains, certaines* | pluriel, accord en genre | pluriel |
| *plusieurs* | pluriel, **invariable** en genre | pluriel |
| *divers, différents* (antéposés) | pluriel | pluriel |
| *chaque* | **singulier obligatoire** | singulier |
| *tout, toute, tous, toutes* (dét.) | selon nom | selon nom |
| *aucun(e)* | **singulier** | singulier |

**Pièges TCF (à mémoriser comme des couples)**

| ✓ Correct | ✘ Erreur fréquente |
|---|---|
| *la plupart des étudiants **sont*** | *la plupart des étudiants **est*** |
| *plusieurs étudiantes* | *plusieures étudiantes* |
| *chaque jour* | *chaques jours* |
| *aucune erreur* | *aucunes erreurs* |
| *tous les dix minutes* | *chaque dix minutes* |
| *tout étonnée* (h muet) | *toute étonnée* |
| *toute surprise* (consonne) | *tout surprise* |
| *un peu de pain* | *quelques pain* / *un peu d'amis* |
| *différents avis* (= plusieurs) | confusion avec *avis différents* (= qui diffèrent) |
| *la plupart* (un mot) | *la plus part* (en trois mots) |

**Antéposition / postposition — *divers, différents, certain, quelconque***
- antéposé : sens d'indéfini de quantité (*différents avis* = plusieurs).
- postposé : sens d'adjectif qualificatif plein (*avis différents* = qui se distinguent ; *succès certain* = sûr ; *raison quelconque* = banale).
