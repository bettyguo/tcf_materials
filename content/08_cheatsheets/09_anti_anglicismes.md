# Anti-anglicismes — 35 calques à bannir

Les **faux-amis** et **calques sémantiques** font chuter la note de lexique en EE/EO. Règle : si le mot ressemble à l'anglais, **vérifiez** avant de l'utiliser. Tableau condensé.

## (a) Les pièges de niveau 1 (fréquents, sanctionnés)

| Anglicisme (✗) | Sens anglais attendu | Naturel français (✓) |
|---|---|---|
| éventuellement | eventually | **finalement**, à terme, par la suite |
| définitivement | definitely | **vraiment**, sans aucun doute, absolument |
| opportunité | opportunity | **occasion**, possibilité |
| réaliser que | to realize that | **se rendre compte que**, comprendre |
| supporter qqn | to support emotionally | **soutenir**, encourager, appuyer |
| application | job application | **candidature**, dossier |
| introduire qqn | to introduce | **présenter** |
| adresser un problème | to address an issue | **aborder**, traiter, s'attaquer à |
| agenda politique | political agenda | **programme**, ordre du jour |
| activer un plan | to activate | **déclencher**, mettre en œuvre |
| impacter | to impact | **avoir un impact sur**, affecter, peser sur |
| en charge de | in charge of | **chargé de**, responsable de |
| au final | in the end | **finalement**, en définitive, au bout du compte |

## (b) Pièges de niveau 2 (subtils, registre soutenu)

| Anglicisme (✗) | Naturel français (✓) |
|---|---|
| consistent (cohérent dans le temps) | **cohérent**, constant, régulier |
| relevant | **pertinent**, approprié |
| engagement (métrique) | **implication**, participation, intérêt |
| flexible | accepté, mais préférer **souple**, adaptable |
| majeur (= principal) | **principal**, central, primordial |
| nominer qqn | **proposer la candidature de**, désigner |
| développer un argument | **approfondir**, étayer, exposer |
| figure (d'autorité) | un **chiffre** = donnée ; pour personnage : **figure** OK, mais préférer **personnalité** |
| processus (surutilisé) | **démarche**, parcours, mécanisme |
| livrable | accepté en gestion de projet, sinon **résultat**, **production** |

## (c) Faux-amis lexicaux notoires

| Mot français | Sens RÉEL | Sens anglais à NE PAS attribuer |
|---|---|---|
| **éducation** | savoir-vivre, valeurs transmises | *education system* → **enseignement, scolarité, formation** |
| **formation** | apprentissage professionnel | ≠ *formation* (= constitution) |
| **lecture** | reading | ≠ *lecture* (= **cours magistral, conférence**) |
| **librairie** | bookshop | ≠ *library* (= **bibliothèque**) |
| **sensible** | sensitive | ≠ *sensible* (= **sensé, raisonnable**) |
| **prétendre** | to claim | ≠ *to pretend* (= **faire semblant**) |
| **assister à** | to attend | ≠ *to assist* (= **aider**) |

## (d) Tournures syntaxiques calquées

- ✗ *Je ne suis pas confortable avec cette idée.* → ✓ *Je ne suis pas **à l'aise** avec cette idée.*
- ✗ *La majorité des gens **pense*** (accord fragile) → ✓ *La **plupart** des gens **pensent***.
- ✗ *C'est rentable d'investir* (calque *it's profitable to*) → ✓ *Investir est **rentable** / il est **rentable d'**investir* (préférer la nominalisation).
- ✗ *Faire du sens* (*to make sense*) → ✓ ***avoir du sens***.
- ✗ *Prendre une décision rapide* (correct), mais ✗ *prendre action* → ✓ ***agir***, prendre des mesures.
- ✗ *Basé sur* (passable, mais surutilisé) → ✓ ***fondé sur***, reposant sur, à partir de.
- ✗ *Au niveau de* (vague) → ✓ ***en matière de***, sur le plan de, concernant.
- ✗ *Versus* en rédaction → ✓ ***par rapport à***, face à, contre.

## (e) Régionalismes québécois à éviter en TCF Canada (registre standard)

Le TCF accepte la norme **internationale** ; en EE/EO, préférer le standard.

| Québécois courant | Standard recommandé |
|---|---|
| céduler (un rendez-vous) | **planifier**, fixer |
| canceller | **annuler** |
| breuvage | **boisson** |
| chum / blonde | **petit ami / petite amie**, conjoint(e) |
| dispendieux | **coûteux**, cher |
| présentement | **actuellement**, en ce moment |

## (f) Mini-stratégie EE/EO

1. Au moment d'écrire un mot transparent (≈ anglais), **doutez** : est-ce le bon sens ?
2. Avez-vous un **synonyme natif** ? Préférez-le (gain en note de lexique).
3. Pour les chiffres : *chiffre / nombre / donnée / statistique* — variez.
4. Pour les verbes faibles (*faire, mettre, avoir*) : substituez **un verbe précis** (*élaborer, instaurer, présenter*).
