# Liaisons — règles de survie

La liaison = prononciation d'une consonne finale normalement muette devant voyelle. **Trois catégories** : obligatoire (à faire toujours), interdite (à ne jamais faire), facultative (selon registre).

## (a) Liaisons OBLIGATOIRES

À l'intérieur d'un **groupe rythmique** étroitement soudé.

| Contexte | Exemple | Liaison |
|---|---|---|
| Déterminant + nom | les_amis, mon_ami, un_homme, des_enfants | /z/, /n/, /n/, /z/ |
| Pronom sujet + verbe | nous_avons, ils_ont, on_a, vous_êtes | /z/, /z/, /n/, /z/ |
| Adj. antéposé + nom | petits_enfants, grand_homme, bon_ami | /z/, /t/, /n/ |
| Préposition courte + nom | en_avion, sans_aide, dans_un, chez_elle | /n/, /z/, /n/, /z/ |
| Verbe + pronom inversé | sont-ils, va-t-on, a-t-elle, peut-on | /t/, /t/, /t/, /t/ |
| Expressions figées | tout_à l'heure, de plus_en plus, pied_à terre | /t/, /z/, /t/ |
| *c'est* + voyelle | c'est_important, c'est_évident | /t/ |

Drill : *Les_anciens_élèves ont_organisé un_événement._*

## (b) Liaisons INTERDITES

Frontière syntaxique forte ou contrainte phonétique.

| Contexte | Exemple | Symbole |
|---|---|---|
| Nom sg. + verbe | le garçon // arrive | // |
| Sujet nominal + verbe | mon ami // est venu | // |
| Conjonction *et* | et // alors, et // ensuite | // (jamais) |
| Devant *h aspiré* | les // héros, en // haut, les // haricots | // |
| Nom sg. + adjectif | un livre // intéressant, un plat // épicé | // |
| Devant chiffre | les // huit livres, les // onze élèves | // |
| Interrogatif + verbe | comment // allez-vous (formel possible) | // |

## (c) Liaisons FACULTATIVES

Choix selon le **registre** : formel/lu → liaison faite ; oral courant → omise.

| Contexte | Soutenu | Courant |
|---|---|---|
| Verbe + complément | je vais_aller | je vais // aller |
| Nom pluriel + adj. | des livres_intéressants | des livres // intéressants |
| Après *pas / plus / jamais* | pas_encore, plus_aucun | pas // encore |
| *est* + adj./part. | il est_arrivé | il est // arrivé |

**Stratégie TCF EO** : viser un registre standard — quelques liaisons facultatives faites (= soigné), mais pas toutes (= pédant). Au-dessus de B2, en faire **plus** que moins.

## (d) Pièges du *h aspiré*

Le *h aspiré* (origine germanique) bloque la liaison ET l'élision. À mémoriser :

| h aspiré (pas de liaison) | h muet (liaison normale) |
|---|---|
| les // héros, le héros | les_hommes, l'homme |
| en // haut, la haine | les_heures, l'heure |
| le hibou, par hasard | l'hiver, les_hivers |
| la honte, les // haricots | l'hôpital, les_hôpitaux |
| le hareng, la hache | l'horizon, l'humanité |

Liste rapide *h aspiré* fréquents : **héros, haut, haine, hibou, hasard, honte, haricot, hareng, hache, hangar, harpe, hibou, hockey, honte, hors**.

Note : **héroïne, héroïque** = *h muet* (l'héroïne) ≠ **héros** = *h aspiré* (le héros).

## (e) Liaisons sonores : 4 sons à distinguer

- **/z/** : finales en *-s*, *-x*, *-z* — les_amis, deux_amis, chez_eux.
- **/t/** : finales en *-t*, *-d* — petit_ami, grand_homme, prend-il.
- **/n/** : finales en *-n* (nasales) — un_ami, mon_ami, en_avion.
- **/ʁ/** : finales en *-r* (rare en liaison vraie ; surtout enchaînement) — premier_étage.

## (f) Liaisons-pièges fréquentes

- *cent_un* (/sɑ̃tœ̃/) mais *cent // huit* (/sɑ̃ɥit/).
- *vingt_ans* avec /t/ (et non /d/ malgré l'écriture).
- *quand_il vient* /kɑ̃til/ — le *d* devient /t/ en liaison.
- *grand_homme* /gʁɑ̃tɔm/ — idem.
- *neuf_ans*, *neuf_heures* → /v/ (et non /f/).
