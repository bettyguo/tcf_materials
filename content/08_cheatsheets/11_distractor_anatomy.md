# Anatomie des distracteurs — version sac d'examen

> Sept catégories exhaustives pour QCM CO + CE. Pour chaque option suspecte : nommer le piège, appliquer la tactique. À glisser dans le sac.

## Tableau de défense

| Catégorie | Tactique |
|---|---|
| **1. Mot-piège** — un mot de la source réapparaît dans l'option, mais dans un autre rapport sémantique. *Ex. : « Le maire rejette la piétonnisation. » → option : « Il soutient la piétonnisation. »* | Ne jamais valider une option parce qu'elle « contient les bons mots ». Vérifier le **verbe** et la **polarité** qui relient ces mots. |
| **2. Généralisation abusive** — une affirmation locale est transformée en thèse universelle. *Ex. : « Dans cette étude-ci… » → option : « Les personnes âgées dorment toujours mieux sans écrans. »* | Repérer les quantificateurs implicites. Méfiance devant *tous, toujours, jamais, partout* ; séparer **thème** et **thèse**. |
| **3. Contraire littéral** — un quantifieur ou une négation est discrètement inversé, le reste du libellé est identique. *Ex. : « Peu de chercheurs contestent… » → option : « Beaucoup de chercheurs contestent… »* | Traiter les mots de **quantité et de négation** comme des mots pleins, pas comme du bruit. À l'oral ils sont peu accentués : redoubler d'attention. |
| **4. Inférence non garantie** — l'option propose une conclusion plausible mais que la source n'étaye pas. *Ex. : « Le télétravail a réduit les trajets. » → option : « Donc il a réduit la pollution. »* | Test : *puis-je pointer la phrase exacte qui force cette conclusion ?* Sinon, c'est une inférence ajoutée. Le TCF demande ce qui est **dit ou nécessairement impliqué**, pas ce qui est probable. |
| **5. Faux degré** — l'intensité ou la modalité est durcie ou affaiblie. *Ex. : « Ces résultats suggèrent un lien. » → option : « Ces résultats prouvent le lien. »* | Calibrer les verbes de modalité : *sembler, suggérer, pourrait* ≠ *prouver, démontrer, garantir*. Un distracteur durcit souvent la prudence en certitude. |
| **6. Attribution glissante** — l'option attribue au locuteur principal une thèse qu'il ne fait que rapporter. *Ex. : « Selon ses détracteurs, la réforme serait précipitée. » → option : « Le ministre estime que la réforme est précipitée. »* | Suivre les marqueurs de **discours rapporté** (*selon, d'après, pour untel, conditionnel journalistique*). Séparer la voix de X des voix qu'il cite. |
| **7. Faux-ami français-anglais** — l'option exploite un cognat dont le sens français diffère de l'anglais. *Ex. : « Le projet sera éventuellement reconduit. » → option : « Le projet sera finalement reconduit. »* | Pour tout cognat transparent, suspendre la lecture anglaise. Vérifier : *éventuellement* = possibly, *définitivement* = for good, *actuellement* = currently, *assister à* = to attend, *prétendre* = to claim. |

## Recouvrement par modalité

- **CO (audio, une écoute)** : pièges 1, 3 et homophones les plus dangereux. Lire les questions pendant la consigne ; viser la **paraphrase**, pas la répétition.
- **CE (texte, relecture)** : pièges 4 et 6 dominants. Localiser le **paragraphe cible** avant d'appliquer la taxonomie aux 4 options.

## Auto-diagnostic après chaque série de 10

Noter pour chaque erreur le **numéro de piège** (1–7). Un profil émerge après 3 séries : si > 40 % de type 4, votre faiblesse est la sur-interprétation, pas le vocabulaire — la remédiation est différente.
