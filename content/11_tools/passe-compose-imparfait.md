# Drill PC vs imparfait

> **Le piège le plus rentable à 5 minutes par jour.** 30 items qui couvrent les 10 patrons qui décident à 95 % entre passé composé et imparfait en français. Why-string sur chaque item — l'explication est le vrai gain.

<div class="tcf-tense" data-set="pc_imp" data-tool="tense"></div>

## Les 10 patrons à intégrer

| Patron | PC | Imparfait |
|---|---|---|
| Action ponctuelle / unique | ✓ | ✗ |
| Action en cours interrompue | (l'action ponctuelle) | (l'action en cours) |
| Habitude / fréquence indéterminée | ✗ | ✓ |
| État mental qui dure | (changement d'état) | (état stable) |
| Décor narratif descriptif | ✗ | ✓ |
| Durée délimitée et achevée | ✓ | ✗ |
| Durée continue / non-clôturée | ✗ | ✓ |
| Nombre déterminé d'occurrences | ✓ | ✗ |
| Pouvoir / vouloir / savoir au sens d'incapacité durable | ✗ | ✓ |
| Pouvoir / vouloir / savoir au sens ponctuel | ✓ | ✗ |

## Astuces de récupération rapide

- **Soudain · tout à coup · à ce moment-là** → presque toujours PC
- **Pendant que · alors que · quand (= au moment où)** + verbe **en cours** → imparfait pour la durée, PC pour l'événement
- **Tous les + jour/semaine/an · souvent · toujours · à l'époque** → imparfait
- **« n'a pas pu » vs « ne pouvait pas »** → PC = incapacité ponctuelle ; imparfait = état durable

## Après ce drill

Ne vous arrêtez pas au score. Pour chaque item raté :

1. **Notez la phrase** dans le [journal d'erreurs](journal.md) avec la forme **correcte** (= la cible) en regard.
2. **Ajoutez la why-string** dans le champ « raison » du journal.
3. **Reprenez** dans 1 j → 3 j → 7 j → 14 j (SRS automatique du journal).
