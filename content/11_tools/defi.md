# Défi quotidien

> **3 minutes par jour, déterministe, partageable.** Cinq questions tirées chaque jour des banques cloze / genre / connecteurs / synonymes / temps. La sélection est **fixée par la date** — tout le monde a le même défi le même jour. Le score s'archive automatiquement.

<div class="tcf-daily" data-tool="daily"></div>

## Pourquoi le défi marche

- **Diversité forcée** : mélange de 5 patrons grammaticaux ou lexicaux différents → réveille les bandes oubliées (vs. une session monothématique qui les laisse rouiller).
- **Engagement court** : 3 minutes maximum, donc compatible avec n'importe quelle journée surchargée. C'est le bloc qui survit aux semaines difficiles.
- **Streak quotidien spécifique** : même si vous n'avez pas le temps pour une session longue, jouer le défi maintient votre série journalière dans la heatmap [streak](streak.md) et le tableau de bord.

## Si vous ratez 2/5 sur la même catégorie

Plongez immédiatement dans l'outil spécialisé :

| Catégorie | Outil de remédiation |
|---|---|
| Genre | [Drill genre des noms](genre.md) |
| Connecteurs | [Drill connecteurs B2](connecteurs.md) |
| Synonymes | [Drill synonymes B2](synonymes.md) |
| PC / imparfait | [Drill PC vs imparfait](passe-compose-imparfait.md) |

Ne réfléchissez pas trop : **5 minutes d'outil ciblé après un défi raté = la session la plus rentable de votre semaine.**

## Note technique

La graine de sélection est `hash(YYYY-MM-DD)`. Concrètement : deux navigateurs ouverts le même jour voient les mêmes 5 questions dans le même ordre. Vous pouvez comparer vos résultats avec un partenaire d'étude sans tricher.
