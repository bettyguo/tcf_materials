# Équivalence CECRL ↔ NCLC

## Notes de lecture

- **Score binding (le minimum des 4)** détermine votre NCLC officiel pour Express Entry. Aucune compétence ne « rattrape » une autre.
- **NCLC 7** = porte d'entrée Catégorie de l'expérience canadienne (CEC) sans bonus CRS.
- **NCLC 8** = **bonus CRS +25** (toutes compétences ≥ 8). Seuil ciblé par ce corpus.
- **NCLC 9** = **bonus CRS +50** si vous avez aussi anglais ≥ NCLC 9. C'est l'objectif maximal réaliste pour la majorité des candidats francophones non-natifs.
- **NCLC 10** = plafond officiel ; au-delà, aucun avantage CRS supplémentaire.

## Bornes CO / CE bruts (rappel)

Les seuils CO et CE sont identiques. Pour passer **B2 (NCLC 8)** à partir d'un score brut /699 :

- **NCLC 8** : 503-522
- **NCLC 9** : 523-548
- **NCLC 10** : 549-699

Pour EE et EO (notés /20) : NCLC 8 = 14-15 · NCLC 9 = 16-17 · NCLC 10 = 18-20.

## Pour vérifier vos scores

Saisissez vos 4 scores bruts dans le [**calculateur NCLC**](calculateur-nclc.md). Vous obtenez les 4 NCLC individuels + le **binding** + le bonus CRS estimé. Lien partageable produit.
