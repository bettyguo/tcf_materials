# Construire la phrase

> **L'ordre des mots est de la grammaire.** Donné un sens (traduction anglaise) ou simplement un défi visuel, recomposez la phrase française correcte à partir de jetons mélangés. C'est l'outil le plus court pour **muscler l'intuition syntaxique** — décisif pour l'EE et l'EO.

<div class="tcf-builder" data-set="starter" data-tool="builder"></div>

## Cibles couvertes (lot 1)

| Phrase | Pivot grammatical |
|---|---|
| Je ne pense pas que ce soit une bonne idée. | Subjonctif après « ne pas penser que » |
| Bien qu'il ait plu, la fête a eu lieu. | « Bien que » + subj. passé |
| Il faut que tu viennes avant midi. | Subjonctif présent obligatoire |
| Si j'avais su, je serais venu plus tôt. | Hypothétique 3 — irréel du passé |
| C'est le livre dont je t'ai parlé hier. | Pronom relatif « dont » (parler de) |
| Plus il lit, plus il comprend. | Structure corrélée |
| En raison des travaux, le trafic est très ralenti. | Locution causale |
| Quoi qu'il dise, personne ne le croira. | Concession indéfinie + subj. |
| Il n'est pas venu parce qu'il était malade. | Cause + indicatif |
| Je ne savais pas qu'elle habitait à Québec. | « habiter à » + ville |

## Méthode

1. Lisez la **traduction-cible** d'abord — elle vous donne le sens, pas l'ordre.
2. Posez **le sujet** en premier, puis **le verbe**, puis les compléments dans l'ordre français (sujet-verbe-complément essentiel, puis adverbiaux).
3. Vérifiez les **mots-outils** (`ne … pas`, `que / qu'`, `dont`, `parce que / parce qu'`) — ils encadrent la structure.
4. Si vous échouez, **refaites immédiatement** la même phrase. Le bénéfice est en l'écart pas-bon → bon, pas dans le score.
