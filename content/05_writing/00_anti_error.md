# Registre anti-erreurs — 56 entrées

> Document **vivant** : chaque erreur listée est observée chez les candidats TCF anglophones en transit B1→B2. Chaque entrée donne la forme fautive, la forme correcte, l'explication, et un renvoi vers l'unité de grammaire qui couvre le mécanisme sous-jacent.

Pour chaque entrée :
- ✗ = forme fautive (ne pas écrire)
- ✓ = forme correcte
- → = explication
- 🔗 = unité grammaire / pivot

---

## A. Subjonctif (~12 entrées) — le piège n° 1 en T3

### A1 — *bien que* + indicatif

- ✗ *Bien que la situation est difficile, …*
- ✓ *Bien que la situation soit difficile, …*
- → *Bien que* déclenche **toujours** le subjonctif, même si la situation est avérée. C'est une **règle structurelle**, pas une question de doute.
- 🔗 [01_grammar/b2_core/](../01_grammar/b2_core/) gram-b2-01

### A2 — *il faut que* + indicatif

- ✗ *Il faut que je fais cela.*
- ✓ *Il faut que je fasse cela.*
- → Toute construction impersonnelle de nécessité ou de souhait (*il faut, il est nécessaire, il est important, il est souhaitable, il vaut mieux*) déclenche le subjonctif.
- 🔗 gram-b2-01

### A3 — *pour que / afin que* + indicatif

- ✗ *Pour que vous comprenez, …*
- ✓ *Pour que vous compreniez, …*
- → Toutes les conjonctions de but (*pour que, afin que*) déclenchent le subjonctif.
- 🔗 gram-b2-01

### A4 — *avant que* + indicatif

- ✗ *Avant que vous partez, …*
- ✓ *Avant que vous (ne) partiez, …*
- → *Avant que* + subj. ; le *ne* explétif (sans valeur négative) est typiquement soutenu.
- 🔗 gram-b2-01

### A5 — *à condition que / à moins que* + indicatif

- ✗ *À condition que tu peux venir, …*
- ✓ *À condition que tu puisses venir, …*
- → Toutes les conjonctions de condition autres que *si* exigent le subj.
- 🔗 gram-b2-01

### A6 — Subjonctif après *douter, regretter, craindre*

- ✗ *Je doute qu'il vient.*
- ✓ *Je doute qu'il vienne.*
- → Verbes d'opinion **négative ou hypothétique** + *que* → subj.
- 🔗 gram-b2-01

### A7 — *espérer que* + subjonctif (calque)

- ✗ *J'espère qu'il vienne.*
- ✓ *J'espère qu'il viendra.*
- → *Espérer que* prend l'**indicatif** (le plus souvent au futur). Ne pas confondre avec *souhaiter que* (+ subj.).
- 🔗 gram-b2-01

### A8 — *penser que* négatif → subj.

- ✗ *Je ne pense pas qu'il a raison.*
- ✓ *Je ne pense pas qu'il ait raison.*
- → *Penser que* à la forme affirmative = indicatif ; à la forme négative ou interrogative = subj. (incertitude).
- 🔗 gram-b2-01

### A9 — Subjonctif après *le seul / le premier / le dernier qui*

- ✗ *C'est le seul qui peut le faire.*
- ✓ *C'est le seul qui puisse le faire.*
- → Relatives après superlatif ou raréfication → subj.
- 🔗 gram-c1-01

### A10 — Subjonctif imparfait : à éviter en EE

- → Le subjonctif imparfait (*qu'il vînt, qu'il fît*) n'est plus actif en français contemporain ; même en C1 EE, on substitue le subj. présent ou la subordonnée à l'indicatif. **Ne pas tenter** en examen.

### A11 — Subjonctif passé après concession (registre soutenu)

- ✓ *Bien qu'il ait fait son maximum, le projet n'a pas abouti.*
- → Subj. passé = action accomplie dans un cadre concessif. Maîtrisé = point C4.
- 🔗 gram-c1-01

### A12 — *quoique* vs *quoi que*

- ✗ *Quoique tu fasses, …*
- ✓ *Quoi que tu fasses, …* (= "n'importe quoi que")
- → *Quoique* (en un mot) = "bien que" (concession). *Quoi que* (en deux mots) = "n'importe quoi que" (relatif indéfini). Les deux prennent le subj. mais ne signifient pas la même chose.

## B. Précision lexicale & calques (~14 entrées)

### B1 — *adresser un problème* (calque)

- ✗ *Il faut adresser ce problème.*
- ✓ *Il faut aborder / traiter / régler ce problème.*
- → *Adresser* en français = envoyer un message à quelqu'un. Pour "to address an issue", utiliser *aborder, traiter, régler*.

### B2 — *opportunité de faire X* (calque)

- ✗ *J'ai eu l'opportunité de visiter…*
- ✓ *J'ai eu l'occasion de visiter…*
- → *Opportunité* en français existe mais reste rare et soutenue (= "caractère opportun"). Pour "opportunity to do", utiliser *occasion* ou *chance*.

### B3 — *initier un projet* (calque)

- ✗ *Cette équipe a initié un projet.*
- ✓ *Cette équipe a lancé / amorcé / entamé un projet.*
- → *Initier* en français = introduire quelqu'un à un savoir, à un rite (initier qqn à qqch).

### B4 — *appliquer pour un poste* (calque)

- ✗ *J'ai appliqué pour ce poste.*
- ✓ *J'ai postulé à / pour ce poste.* ou *J'ai candidaté à ce programme.*
- → *Appliquer* = poser sur, mettre en pratique. Pas un verbe de candidature.

### B5 — *compléter un formulaire* (calque)

- ✗ *Veuillez compléter ce formulaire.*
- ✓ *Veuillez remplir ce formulaire.*
- → *Compléter* = ajouter ce qui manque (à un ensemble déjà partiel). *Remplir* = écrire dans toutes les cases d'un formulaire.

### B6 — *réaliser que* (anglicisme toléré)

- ⚠ *J'ai réalisé que…*
- ✓ *Je me suis rendu compte que… / J'ai pris conscience que…*
- → *Réaliser que* est attesté en français contemporain mais marqué. En T3 soutenu, préférer la forme classique.

### B7 — *définitivement* ≠ *definitely*

- ✗ *C'est définitivement vrai.* (pour "definitely")
- ✓ *C'est absolument vrai. / C'est tout à fait vrai.*
- → *Définitivement* = pour de bon, irrémédiablement (*il est parti définitivement*).

### B8 — *éventuellement* ≠ *eventually*

- ✗ *Éventuellement, il a réussi.* (pour "eventually")
- ✓ *Finalement, il a réussi.*
- → *Éventuellement* = "possibly" (hypothèse). *Eventually* = finalement, à terme.

### B9 — *supporter* ≠ *to support*

- ✗ *Je supporte cette idée.*
- ✓ *Je soutiens cette idée. / J'appuie cette idée.*
- → *Supporter* = endurer, tolérer (et, sportif, "supporter une équipe"). Pour "to support an idea", utiliser *soutenir / appuyer*.

### B10 — *prendre une décision* vs *faire une décision*

- ✗ *Faire une décision*
- ✓ *Prendre une décision*
- → Calque de *to make a decision*. Le verbe correct est *prendre*.

### B11 — *avoir du sens* vs *faire du sens*

- ✗ *Cette idée fait du sens.*
- ✓ *Cette idée a du sens. / Cette idée se tient.*
- → *Faire du sens* est un anglicisme québécois courant à l'oral. En EE TCF (registre standard), bannir.

### B12 — *introduire qqn à qqn* vs *présenter qqn à qqn*

- ✗ *Je vous introduis Paul.*
- ✓ *Je vous présente Paul.*
- → *Introduire* = faire entrer matériellement. *Présenter* = faire connaître à.

### B13 — *davantage de* vs *plus de*

- ✓ Les deux sont corrects ; *davantage de* est plus soutenu et moins fréquent.
- → Préférer *davantage de* en T3 pour C3.

### B14 — *notamment* vs *en particulier* — non interchangeables

- *Notamment* : marque une **énumération non exhaustive** (le X cité parmi d'autres). *Notamment Paul, Marie et Jean…*
- *En particulier* : **spécifie** un cas dans un ensemble plus large. *Tous, en particulier Paul, …*
- → Sur copie soutenue, l'usage incorrect coûte 1 point sur C3.

## C. Connecteurs et registre (~8 entrées)

### C1 — *ainsi* en début de phrase → inversion possible

- ✓ *Ainsi a-t-il décidé de partir.* (registre soutenu, inversion sujet-verbe)
- → L'inversion après *ainsi, à peine, peut-être, sans doute* en tête de phrase est un marqueur C1 fort.
- 🔗 gram-c1-01 inversion stylistique

### C2 — *cependant* vs *par contre*

- *Par contre* : registre **courant**, parfois marqué comme familier en France (Académie réticente).
- *Cependant, néanmoins, en revanche* : registre soutenu ; en T2/T3 formel, préférer ces formes.
- → *En revanche* exige une opposition entre deux choses **comparables** (*Pierre est petit ; en revanche, Paul est grand*).

### C3 — *aussi* en tête de phrase

- *Aussi est-il parti.* = "C'est pourquoi il est parti." (registre soutenu, inversion).
- *Il est aussi parti.* = en plus de qqn d'autre (registre courant).
- → Confusion fréquente ; vérifier que l'inversion accompagne *aussi* conséquent.

### C4 — *malgré que* (souvent fautif)

- ⚠ *Malgré que* + subj. existe à l'oral mais est **condamné par l'Académie**. En T3 EE, préférer *bien que* + subj.
- → *Malgré* + nom : *malgré le froid* ✓.

### C5 — *en effet* ≠ *donc*

- ✗ *Il pleut, en effet je reste chez moi.* (sens : "donc")
- ✓ *Il pleut. C'est pourquoi je reste chez moi.* ou *Il pleut, donc je reste chez moi.*
- → *En effet* = "indeed, confirming what was said". Pas de valeur conséquentielle.

### C6 — *par ailleurs* vs *d'ailleurs*

- *Par ailleurs* : introduit un **argument additionnel** non lié au précédent (registre soutenu).
- *D'ailleurs* : confirme et renforce **ce qu'on vient de dire** (registre courant à soutenu).
- → Confusion coûteuse sur C2.

### C7 — *finalement* vs *enfin*

- *Finalement* = au bout du compte (résultat).
- *Enfin* = en dernier lieu (ordre énumératif).
- → *D'abord, ensuite, enfin* (énumération). *Finalement, il a accepté* (résultat).

### C8 — *or* en début de phrase

- *Or* introduit un **élément qui change la donne** (factuel ou logique). Registre soutenu, C1.
- ✓ *On pensait que X ; or, les chiffres montrent Y.*
- → Pas un synonyme de *donc* ni de *cependant*. À utiliser en T3 pour un effet C1 net.

## D. Accords (~8 entrées)

### D1 — Accord du PP avec *avoir* + COD antéposé

- ✗ *Les fleurs que j'ai cueilli sont fanées.*
- ✓ *Les fleurs que j'ai cueillies sont fanées.*
- → Avec *avoir*, le PP s'accorde avec le COD seulement si celui-ci est placé **avant** le verbe. Erreur la plus fréquente, et la plus pénalisée.
- 🔗 gram-b2-04

### D2 — Accord avec *être*

- ✓ *Elle est partie. / Elles sont parties.*
- → Avec *être*, le PP s'accorde toujours avec le sujet.

### D3 — *tout* — adverbe (invariable) vs adjectif (variable)

- *Tout content* (adv. devant adj. masc.) — invariable.
- *Toute contente* (adv. devant adj. fém. commençant par une consonne ou h aspiré) — accord apparent.
- *Toute la maison* (adj.) — accord normal.
- → Souvent piège d'examen.

### D4 — *quelque* — déterminant variable vs adverbe invariable

- *quelques amis* (det) ✓
- *quelque cent personnes* (adv. = "environ", invariable) ✓
- → Confusion fréquente en T3.

### D5 — Accord de *vingt* et *cent*

- ✗ *Quatre-vingt euros*
- ✓ *Quatre-vingts euros* (s quand non suivi d'un autre numéral)
- ✓ *Quatre-vingt-deux euros* (sans s car suivi de *deux*)
- → *Cent* suit la même règle. *Deux cents euros* ; *deux cent dix euros*.

### D6 — *demi*

- ✓ *Une demi-heure.* (avant le nom, invariable, avec trait d'union)
- ✓ *Une heure et demie.* (après le nom, accord)

### D7 — Sujet inversé : accord avec le vrai sujet

- ✗ *Il y a des erreurs qui ont été commis.*
- ✓ *Il y a des erreurs qui ont été commises.*
- → Le PP s'accorde avec *erreurs* (sujet de *ont été commises*).

### D8 — *Madame* / *Monsieur* en tête de courrier — virgule obligatoire

- ✓ *Madame,* (virgule, et *non* le nom)
- → Erreur fréquente en T1 ; perte de point sur C1 si formule d'appel non normée.

## E. Constructions verbales (~7 entrées)

### E1 — *se rappeler* (transitif direct) vs *se souvenir de*

- ✗ *Je me rappelle de cette histoire.*
- ✓ *Je me rappelle cette histoire.* / *Je me souviens de cette histoire.*
- → *Se rappeler* est transitif direct. *Se souvenir* prend *de*. Confondu très souvent.

### E2 — *demander* + n / + à + inf

- ✗ *J'ai demandé de venir.* (ambigu)
- ✓ *J'ai demandé à venir.* (= je veux venir) ; *J'ai demandé de venir* (= j'ai demandé à qqn de venir).
- → *Demander à* + inf = je veux faire X. *Demander de* + inf = j'ai demandé à qqn de faire X.

### E3 — *jouer à* (sport) vs *jouer de* (instrument)

- ✓ *Je joue au football.* / *Je joue du piano.*
- → Préposition fixe.

### E4 — *s'agir de* — impersonnel

- ✗ *Je m'agis de…*
- ✓ *Il s'agit de…* (toujours impersonnel)

### E5 — *commencer à* + inf vs *commencer par* + inf

- ✓ *J'ai commencé à apprendre le français.* (début de l'action)
- ✓ *J'ai commencé par apprendre l'alphabet.* (étape liminaire dans une séquence)

### E6 — *finir de* + inf vs *finir par* + inf

- ✓ *J'ai fini de manger.* (achever)
- ✓ *J'ai fini par accepter.* (à la fin)

### E7 — *manquer à* (cause de manque) vs *manquer de* (absence partielle)

- ✓ *Cet ami me manque.* (cause de manque — sujet et objet inversés par rapport à l'anglais)
- ✓ *Je manque de temps.* (absence)
- → *Cet ami me manque* ≠ "I miss this friend" (calque) ; structurellement, *moi* est COI.

## F. Registre & politesse (~5 entrées)

### F1 — Pronom *on* en T3

- ✓ *On voit ainsi que…* (acceptable mais pas valorisé en T3)
- ✓ Mieux : *Force est de constater que…* / passif / *il convient de noter que*
- → *On* abuse en T3 fait paraître la copie "rédactionnelle scolaire". En T2 témoignage, acceptable.

### F2 — *Madame, Monsieur* vs *Monsieur, Madame*

- ✓ *Madame, Monsieur,* (ordre canonique en France pour un destinataire inconnu)
- → Erreur fréquente sur l'ordre.

### F3 — *Je vous prie d'agréer …*

- ✓ *Je vous prie d'agréer, Madame, Monsieur, l'expression de mes salutations distinguées.*
- → Formule canonique très soutenue. Ne PAS abréger ni inventer.

### F4 — *Cordialement* — neutre soutenu

- ✓ *Cordialement,* convient pour un échange professionnel sans intimité.
- ✓ *Bien à vous* convient en registre chaleureux mais professionnel (collègue ou interlocuteur connu).

### F5 — *à bientôt* — clôture informelle

- ⚠ *À bientôt* convient à T1 *familier* mais pas à T1 *formel*. Vérifier `register_required` du sujet.

## G. Quebec vs France (~4 entrées) — pour candidat à Montréal

Pour TCF Canada, le registre attendu en EE est le **français standard de France**. Les lexiques québécois sont compris mais ne valorisent pas le score (et peuvent être pénalisés si jugés non-standard par un correcteur métropolitain).

### G1 — *magasiner* / *faire du shopping* / *faire des courses*

- ✓ (France) *faire les courses* (aliments), *faire du shopping* (vêtements, plaisir)
- ⚠ (Québec) *magasiner* — éviter en EE.

### G2 — *char* (Québec) / *voiture* (standard)

- ✓ *une voiture / un véhicule*
- ⚠ *un char* — éviter.

### G3 — *cellulaire* (Québec) / *téléphone portable* (France) / *portable*

- ✓ *un téléphone portable / un mobile* — passe partout.
- ⚠ *un cellulaire* — éviter en EE.

### G4 — *déjeuner* — sens

- En France : *déjeuner* = repas de midi.
- Au Québec : *déjeuner* = petit-déjeuner.
- → En EE, suivre la sémantique de France : *petit-déjeuner / déjeuner / dîner*.

## Procédure de relecture (5 min avant rendu)

Pour rattraper 1 à 2 points de score perdus par inattention, parcourir la copie avec cette grille :

1. **Subjonctif** : chaque *bien que, il faut que, pour que, afin que, avant que* → verbe au subj. ?
2. **Accord PP** : chaque verbe au passé composé avec *avoir* → COD avant le verbe ?
3. **Tu/vous** (T1) : un seul registre tenu ?
4. **Calques** : *adresser un problème, opportunité de, appliquer pour* → reformuler.
5. **Sur-pivots** : pas plus de 4 phrases-pivots du même registre dans la copie.
